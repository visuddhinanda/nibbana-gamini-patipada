### ရှင်သေမင်း မမြင်နိုင်သူ

သုညတော လောကံ အဝေက္ခဿု၊ မောဃရာဇ သဒါ သတော။<br>အတ္တာနုဒိဋ္ဌိံ ဦဟစ္စ၊ ဧဝံ မစ္စုတရော သိယာ။

ဧဝံ လောကံ အဝေက္ခန္တံ၊ မစ္စုရာဇာ န ပဿတိ။ (ခု၊၁၊၄၄၈။ စူဠနိ၊၁၈၀။)

မောဃရာဇ = ဒါယကာတော် မောဃရာဇရသေ့။ 
သဒါ = နေ့ရောညဉ့်ပါ အခါခပ်သိမ်း။ 
သတော =သတိပဋ္ဌာန် လေးပါး မပြတ်ပွားလျက် ထင်သော သတိနှင့် ပြည့်စုံသည် ဖြစ်၍။ 
အတ္တာနုဒိဋ္ဌိံ = အတ္တဟူ၍ အစဉ်ပြေးသွားသော နှစ်ဆယ်သော တည်ရာဝတ္ထု ရှိသော သက္ကာယဒိဋ္ဌိကို။ 
ဦဟစ္စ = နာမရူပပရိစ္ဆေဒဉာဏ် ပစ္စယပရိဂ္ဂဟဉာဏ်တို့ဖြင့် နုတ်ပစ်ပယ်ခွာ၍။ 
သုညတော = နိစ္စ သုခ အတ္တ သုဘစသော အနှစ်သာရမှ ဆိတ်သုဉ်းသောအားဖြင့်။ 
လောကံ = ဒုက္ခသစ္စာ သမုဒယသစ္စာဟူသည့် ရုပ်နာမ်ဓမ္မ သင်္ခါရလောကကို။ 
အဝေက္ခဿု = ဝိပဿနာဉာဏ်ပညာ စက္ခုဖြင့် စေ့စေ့ငုငု ကြည့်ရှုဆင်ခြင်လျက် အဖန်ဖန် အထပ်ထပ် ရှုပွား သုံးသပ်ပါလေလော။ 
ဧဝံ = ဤသို့ ရှုပွား သုံးသပ်နိုင်ခဲ့သည်ရှိသော်။ 
မစ္စုတရော = (၃၁)နိုင်ငံ ပိုင်စံစိုးနင်း ရှင်သေမင်းကို ကျော်လွှားလွန်မြောက်၍ သွားနိုင်သည်။ 
သိယာ = ဖြစ်လေရာ၏။ 
ဧဝံ = ဤသို့ ဟောအပ်ပြီးသော နိစ္စ သုခ အတ္တ ဓုဝ သဿတ အဝိပရိဏာမဓမ္မစသော အနှစ်သာရမှ ဆိတ်သုဉ်းသော အခြင်းအရာအားဖြင့်။ 
လောကံ =ဒုက္ခသစ္စာ သမုဒယသစ္စာဟူသည့် ရုပ်နာမ်ဓမ္မ သင်္ခါရလောကကို။ 
အဝေက္ခန္တံ = ဝိပဿနာဉာဏ်ပညာ စက္ခုဖြင့်စေ့စေ့ငုငု မြော်ရှုဆင်ခြင်လျက်ရှိသော သူကို။ 
မစ္စုရာဇာ = (၃၁)နိုင်ငံ ပိုင်စံစိုးနင်း ရှင်သေမင်းသည်။ 
န ပဿတိ= ဘယ်တော့မှ မမြင်နိုင်လေပြီ။ (ခု၊၁၊၄၄၈။ စူဠနိ၊၁၈၀။ ဝိသုဒ္ဓိ၊၂၊၂၉၃။)
