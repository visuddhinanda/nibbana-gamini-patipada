### ပဋိသန္ဓေ - သင်္ခါရက္ခန္ဓာ၌ ဥဒယ-ဝယရှုပုံ (ဒုတိယနည်း)
**(သမုဒယဝယဓမ္မာနုပဿီ ဝါ ဓမ္မေသု ဝိဟရတိ။)**

- ၁။ အဝိဇ္ဇာ ဖြစ်ခြင်းကြောင့် သင်္ခါရ ဖြစ်၏။
  - အဝိဇ္ဇာ ချုပ်ခြင်းကြောင့် သင်္ခါရ ချုပ်၏။
  - အဝိဇ္ဇာ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - သင်္ခါရ ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၂။ တဏှာ ဖြစ်ခြင်းကြောင့် သင်္ခါရ ဖြစ်၏။
  - တဏှာ ချုပ်ခြင်းကြောင့် သင်္ခါရ ချုပ်၏။
  - တဏှာ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - သင်္ခါရ ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၃။ ဥပါဒါန် ဖြစ်ခြင်းကြောင့် သင်္ခါရ ဖြစ်၏။
  - ဥပါဒါန် ချုပ်ခြင်းကြောင့် သင်္ခါရ ချုပ်၏။
  - ဥပါဒါန် ---- ဖြစ်-ပျက် - အနိစ္စ။
  - သင်္ခါရ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၄။ သင်္ခါရ ဖြစ်ခြင်းကြောင့် (စေတသိက်) သင်္ခါရ ဖြစ်၏။
  - သင်္ခါရ ချုပ်ခြင်းကြောင့် (စေတသိက်) သင်္ခါရ ချုပ်၏။
  - သင်္ခါရ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - (စေတသိက်) သင်္ခါရ ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၅။ ကံ ဖြစ်ခြင်းကြောင့် သင်္ခါရ ဖြစ်၏။
  - ကံ ချုပ်ခြင်းကြောင့် သင်္ခါရ ချုပ်၏။
  - ကံ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - သင်္ခါရ ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၆။ ဝတ္ထု ဖြစ်ခြင်းကြောင့် သင်္ခါရ ဖြစ်၏။
  - ဝတ္ထု ချုပ်ခြင်းကြောင့် သင်္ခါရ ချုပ်၏။
  - ဝတ္ထု ---- ဖြစ်-ပျက် - အနိစ္စ။
  - သင်္ခါရ ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၇။ အာရုံ ဖြစ်ခြင်းကြောင့် သင်္ခါရ ဖြစ်၏။
  - အာရုံ ချုပ်ခြင်းကြောင့် သင်္ခါရ ချုပ်၏။
  - အာရုံ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - သင်္ခါရ ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၈။ ကျန် နာမ်ခန္ဓာ သုံးပါး ဖြစ်ခြင်းကြောင့် သင်္ခါရ ဖြစ်၏။
  - ကျန် နာမ်ခန္ဓာ သုံးပါး ချုပ်ခြင်းကြောင့် သင်္ခါရ ချုပ်၏။
  - ကျန် နာမ်ခန္ဓာ သုံးပါး ---- ဖြစ်-ပျက် - အနိစ္စ။
  - သင်္ခါရ ---- ဖြစ်-ပျက် - အနိစ္စ။
