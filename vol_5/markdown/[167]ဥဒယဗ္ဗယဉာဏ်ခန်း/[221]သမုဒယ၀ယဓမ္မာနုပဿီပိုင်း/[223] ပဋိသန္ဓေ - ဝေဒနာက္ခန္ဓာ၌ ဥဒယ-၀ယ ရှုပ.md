### ပဋိသန္ဓေ - ဝေဒနာက္ခန္ဓာ၌ ဥဒယ-ဝယ ရှုပုံ

**(သမုဒယဝယဓမ္မာနုပဿီ ဝါ ဝေဒနာသု ဝိဟရတိ။)**

- ၁။ အဝိဇ္ဇာ ဖြစ်ခြင်းကြောင့် ဝေဒနာ ဖြစ်၏။
  - အဝိဇ္ဇာ ချုပ်ခြင်းကြောင့် ဝေဒနာ ချုပ်၏။
  - အဝိဇ္ဇာ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝေဒနာ ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၂။ တဏှာ ဖြစ်ခြင်းကြောင့် ဝေဒနာ ဖြစ်၏။
  - တဏှာ ချုပ်ခြင်းကြောင့် ဝေဒနာ ချုပ်၏။
  - တဏှာ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝေဒနာ ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၃။ ဥပါဒါန် ဖြစ်ခြင်းကြောင့် ဝေဒနာ ဖြစ်၏။
  - ဥပါဒါန် ချုပ်ခြင်းကြောင့် ဝေဒနာ ချုပ်၏။
  - ဥပါဒါန် ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝေဒနာ ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၄။ သင်္ခါရ ဖြစ်ခြင်းကြောင့် ဝေဒနာ ဖြစ်၏။
  - သင်္ခါရ ချုပ်ခြင်းကြောင့် ဝေဒနာ ချုပ်၏။
  - သင်္ခါရ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝေဒနာ ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၅။ ကံ ဖြစ်ခြင်းကြောင့် ဝေဒနာ ဖြစ်၏။
  - ကံ ချုပ်ခြင်းကြောင့် ဝေဒနာ ချုပ်၏။
  - ကံ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝေဒနာ ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၆။ ဝတ္ထု ဖြစ်ခြင်းကြောင့် ဝေဒနာ ဖြစ်၏။
  - ဝတ္ထု ချုပ်ခြင်းကြောင့် ဝေဒနာ ချုပ်၏။
  - ဝတ္ထု ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝေဒနာ ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၇။ အာရုံ ဖြစ်ခြင်းကြောင့် ဝေဒနာ ဖြစ်၏။
  - အာရုံ ချုပ်ခြင်းကြောင့် ဝေဒနာ ချုပ်၏။
  - အာရုံ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝေဒနာ ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၈။ ဖဿ ဖြစ်ခြင်းကြောင့် ဝေဒနာ ဖြစ်၏။
  - ဖဿ ချုပ်ခြင်းကြောင့် ဝေဒနာ ချုပ်၏။
  - ဖဿ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝေဒနာ ---- ဖြစ်-ပျက် - အနိစ္စ။

ဤပဋိသန္ဓေ (ဘဝင်-စုတိ) တို့၏ အာရုံမှာ ကံ-ကမ္မနိမိတ်-ဂတိနိမိတ်ဟူသော အာရုံ သုံးမျိုးတို့တွင်တစ်မျိုးမျိုး ဖြစ်၏။ 
ထိုတွင် ကံမှာ လူသားများအတွက် ကုသိုလ်စေတနာတည်း။ 
ဂတိနိမိတ်မှာလည်း ရူပါရုံဟုပင်သတ်မှတ်ချက်ရှိ၏။ (အဘိ၊ဋ္ဌ၊၂၊၁၄၉။) 
ကမ္မနိမိတ်မှာ ပညတ် ပရမတ် ထိုက်သည့်အားလျော်စွာ ဖြစ်ဖွယ်ရာရှိသည်။ 
ယင်းကမ္မနိမိတ်မှာ ဘုရားစေတီတော်ဖြစ်လျှင် ဘုရားစေတီတော်၌ တည်ရှိသော ဓာတ်ကြီး (၄)ပါးကိုစိုက်၍ ရှုလိုက်ပါက ယခုကဲ့သို့ ဥဒယဗ္ဗယဉာဏ်ပိုင်းသို့ ရောက်ရှိနေသော ယောဂါဝစရပုဂ္ဂိုလ်သူမြတ်၏ ဉာဏ်ဝယ်ရုပ်ကလာပ်အမှုန်များကိုသာ မကြာမီ တွေ့ရှိမည် ဖြစ်၏။ 
ယင်းရုပ်ကလာပ်တို့မှာ ဥတုဇဩဇဋ္ဌမကရုပ်တို့သာဖြစ်ကြ၏။ 
ရုပ်ကလာပ် အသီးအသီး၌ တည်ရှိသော ယင်းဥတုဇဩဇဋ္ဌမက ရုပ်တရားတို့ကို သိမ်းဆည်းလျက်ယင်းပရမတ် ရုပ်တရားတို့ကို လက္ခဏာယာဉ်တင်၍ ဝိပဿနာရှုပါ။ 
ပန်း ဆီမီး စသော လှူဖွယ်ဝတ္ထု ကမ္မနိမိတ်၌လည်း နည်းတူပင် ရှုပါ။ 
အကယ်၍ ယင်းကမ္မနိမိတ်မှာ အလှူခံပုဂ္ဂိုလ်များ ဖြစ်ကြသည့် ရဟန်းတော်များစသည့် သက်ရှိသတ္တဝါတို့ ဖြစ်ကြလျှင် ယင်း အလှူခံပုဂ္ဂိုလ် များထိုထိုတို့၏ ရုပ်နာမ်သန္တာန်အစဉ်၌ တည်ရှိသောဓာတ်ကြီးလေးပါးကို ဝိပဿနာဉာဏ်ဖြင့် မြင်အောင် ရှုပါ။ 
ရုပ်ကလာပ်များကို တွေ့ရှိသောအခါ (၆)ဒွါရ(၄၂)ကောဋ္ဌာသတို့၌ တည်ရှိကြသော ရုပ်ပရမတ်တရားတို့ကို ပေါင်းစုခြုံငုံ သိမ်းဆည်း၍ ယင်းပရမတ်တရားတို့ကိုလက္ခဏာယာဉ်တင်ကာ ဝိပဿနာ ရှုပါ။ 
ယင်းကမ္မနိမိတ်ဖြစ်သော အလှူခံ ပုဂ္ဂိုလ်တို့၏ သန္တာန်၌ ဖြစ်ပွားနေကြသော နာမ်တရားများကိုကား သာဝကတို့၏ ဝိပဿနာဉာဏ်က အတိအကျ သိအောင် မရှုနိုင်။ 
ပရစိတ္တဝိဇာနနအဘိညာဏ် ဒိဗ္ဗစက္ခုအဘိညာဏ်တို့၏ အရာသာဖြစ်၏။ 
ထိုကြောင့် ယင်းဗဟိဒ္ဓနာမ်တရားတို့ကို နာမ်တရားချင်းတူညီ၍ သာမညအားဖြင့် ပေါင်းစုခြုံငုံ၍သာ ဝိပဿနာ ရှုပါ။ 
သညာက္ခန္ဓာ သင်္ခါရက္ခန္ဓာ တရားစုတို့ကို အဦးမူ၍ဝိပဿနာရှုခြင်းသည် ဓမ္မာနုပဿနာ သတိပဋ္ဌာန်၌ အကျုံးဝင်၏။ 
ဤတွင် သညာက္ခန္ဓာနှင့် သင်္ခါရက္ခန္ဓာပထမနည်း ရှုကွက်ကို ဝေဒနာက္ခန္ဓာ ရှုကွက်အတိုင်း ရှုပါ။ 
သင်္ခါရက္ခန္ဓာ ဒုတိယနည်း ရှုကွက်ကို ဆက်လက်၍ရေးသားတင်ပြအပ်ပါသည်။ ----
