### ပဋိသန္ဓေ - ဝိညာဏက္ခန္ဓာ၌ ဥဒယ-ဝယရှုပုံ

**(သမုဒယဝယဓမ္မာနုပဿီ ဝါ စိတ္တသ္မိံ ဝိဟရတိ။)**

- ၁။ အဝိဇ္ဇာ ဖြစ်ခြင်းကြောင့် ဝိညာဏ် ဖြစ်၏။
  - အဝိဇ္ဇာ ချုပ်ခြင်းကြောင့် ဝိညာဏ် ချုပ်၏။
  - အဝိဇ္ဇာ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝိညာဏ် ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၂။ တဏှာ ဖြစ်ခြင်းကြောင့် ဝိညာဏ် ဖြစ်၏။
  - တဏှာ ချုပ်ခြင်းကြောင့် ဝိညာဏ် ချုပ်၏။
  - တဏှာ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝိညာဏ် ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၃။ ဥပါဒါန် ဖြစ်ခြင်းကြောင့် ဝိညာဏ် ဖြစ်၏။
  - ဥပါဒါန် ချုပ်ခြင်းကြောင့် ဝိညာဏ် ချုပ်၏။
  - ဥပါဒါန် ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝိညာဏ် ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၄။ သင်္ခါရ ဖြစ်ခြင်းကြောင့် ဝိညာဏ် ဖြစ်၏။
  - သင်္ခါရ ချုပ်ခြင်းကြောင့် ဝိညာဏ် ချုပ်၏။
  - သင်္ခါရ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝိညာဏ် ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၅။ ကံ ဖြစ်ခြင်းကြောင့် ဝိညာဏ် ဖြစ်၏။
  - ကံ ချုပ်ခြင်းကြောင့် ဝိညာဏ် ချုပ်၏။
  - ကံ ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝိညာဏ် ---- ဖြစ်-ပျက် - အနိစ္စ။

- ၆။ နာမ်-ရုပ် ဖြစ်ခြင်းကြောင့် ဝိညာဏ် ဖြစ်၏။
  - နာမ်-ရုပ် ချုပ်ခြင်းကြောင့် ဝိညာဏ် ချုပ်၏။
  - နာမ်-ရုပ် ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဝိညာဏ် ---- ဖြစ်-ပျက် - အနိစ္စ။
  - ဒုက္ခနှင့် အနတ္တတို့ကိုလည်း နည်းတူပင် ဆက်ရှုပါ။
