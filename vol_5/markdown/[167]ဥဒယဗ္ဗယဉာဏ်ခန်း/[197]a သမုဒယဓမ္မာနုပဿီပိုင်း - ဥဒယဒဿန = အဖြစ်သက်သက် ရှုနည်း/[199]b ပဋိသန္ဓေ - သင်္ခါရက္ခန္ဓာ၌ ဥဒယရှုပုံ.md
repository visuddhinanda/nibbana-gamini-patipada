### ပဋိသန္ဓေ - သင်္ခါရက္ခန္ဓာ၌ ဥဒယရှုပုံ (ပထမနည်း)

- ၁။ အဝိဇ္ဇာ (၂၀) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သင်္ခါရ ဖြစ်၏။
- ၂။ တဏှာ (၂၀) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သင်္ခါရ ဖြစ်၏။
- ၃။ ဥပါဒါန် (၂၀) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သင်္ခါရ ဖြစ်၏။
- ၄။ သင်္ခါရ (၃၄) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သင်္ခါရ ဖြစ်၏။
- ၅။ ကံ (= သင်္ခါရ-၃၄-၏ ကမ္မသတ္တိ) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သင်္ခါရ ဖြစ်၏။
- ၆။ ဝတ္ထု (= ဟဒယဝတ္ထု) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သင်္ခါရ ဖြစ်၏။
- ၇။ အာရုံ (= ပဋိသန္ဓေ၏ အာရုံ) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သင်္ခါရ ဖြစ်၏။
- ၈။ ဖဿ (= ၃၄ - စေတနာ = ၃၃) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သင်္ခါရ ဖြစ်၏။

\* (ပဋိသန္ဓေ) သင်္ခါရ၏ ဖြစ်ခြင်းသဘော။

ဤနည်း၌ ပဋိသန္ဓေ ဝိညာဏ်နှင့် ယှဉ်သော စေတနာကိုသာ သင်္ခါရက္ခန္ဓာ-အရ ကောက်ယူထားသည်၊ထိုကြောင့် အကြောင်းတရားဘက်၌ စေတနာမှ ကြွင်းသော နာမ်တရား (၃၃) ဖြစ်သည်။
