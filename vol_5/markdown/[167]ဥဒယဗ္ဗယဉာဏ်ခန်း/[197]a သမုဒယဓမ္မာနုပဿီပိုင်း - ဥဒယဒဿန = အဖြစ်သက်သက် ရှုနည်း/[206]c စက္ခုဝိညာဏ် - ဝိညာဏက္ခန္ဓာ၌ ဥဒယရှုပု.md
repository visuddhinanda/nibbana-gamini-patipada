### စက္ခုဝိညာဏ် - ဝိညာဏက္ခန္ဓာ၌ ဥဒယရှုပုံ

- ၁။ အဝိဇ္ဇာ (၂၀) ဖြစ်ခြင်းကြောင့် စက္ခုဝိညာဏ် ဖြစ်၏။
- ၂။ တဏှာ (၂၀) ဖြစ်ခြင်းကြောင့် စက္ခုဝိညာဏ် ဖြစ်၏။
- ၃။ ဥပါဒါန် (၂၀) ဖြစ်ခြင်းကြောင့် စက္ခုဝိညာဏ် ဖြစ်၏။
- ၄။ သင်္ခါရ (၃၄) ဖြစ်ခြင်းကြောင့် စက္ခုဝိညာဏ် ဖြစ်၏။
- ၅။ ကံ (= သင်္ခါရ-၃၄-၏ ကမ္မသတ္တိ) ဖြစ်ခြင်းကြောင့် စက္ခုဝိညာဏ် ဖြစ်၏။
- ၆။ နာမ်-ရုပ် ဖြစ်ခြင်းကြောင့် စက္ခုဝိညာဏ် ဖြစ်၏။
- ၇။ အာလောက (= အရောင်အလင်း) ဖြစ်ခြင်းကြောင့် စက္ခုဝိညာဏ် ဖြစ်၏။
- ၈။ မနသိကာရ (= ပဉ္စဒွါရာဝဇ္ဇန်း = ၁၁) ဖြစ်ခြင်းကြောင့် စက္ခုဝိညာဏ် ဖြစ်၏။

\* စက္ခုဝိညာဏ်၏ ဖြစ်ခြင်းသဘော။

နာမ် = ယှဉ်ဖက် စေတသိက် (၇) လုံးတည်း။<br>ရုပ် = စက္ခုဒွါရ၌ ရှိသော မှီရာ ဝတ္ထုရုပ် (၅၄) နှင့် ရူပါရုံတည်း။
