### ပဋိသန္ဓေ - သညာက္ခန္ဓာ၌ ဥဒယရှုပုံ

- ၁။ အဝိဇ္ဇာ (၂၀) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သညာ ဖြစ်၏။
- ၂။ တဏှာ (၂၀) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သညာ ဖြစ်၏။
- ၃။ ဥပါဒါန် (၂၀) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သညာ ဖြစ်၏။
- ၄။ သင်္ခါရ (၃၄) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သညာ ဖြစ်၏။
- ၅။ ကံ (= သင်္ခါရ-၃၄-၏ ကမ္မသတ္တိ) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သညာ ဖြစ်၏။
- ၆။ ဝတ္ထု (= ဟဒယဝတ္ထု) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သညာ ဖြစ်၏။
- ၇။ အာရုံ (= ပဋိသန္ဓေ၏ အာရုံ) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သညာ ဖြစ်၏။
- ၈။ ဖဿ (= ၃၄ - သညာ = ၃၃) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) သညာ ဖြစ်၏။
  - \* (ပဋိသန္ဓေ) သညာ၏ ဖြစ်ခြင်းသဘော။
