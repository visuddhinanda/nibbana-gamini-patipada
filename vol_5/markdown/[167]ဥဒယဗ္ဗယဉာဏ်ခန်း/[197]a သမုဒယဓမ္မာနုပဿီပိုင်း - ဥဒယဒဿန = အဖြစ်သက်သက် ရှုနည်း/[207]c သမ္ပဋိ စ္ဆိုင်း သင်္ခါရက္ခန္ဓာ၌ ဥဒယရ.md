### သမ္ပဋိ စ္ဆိုင်း သင်္ခါရက္ခန္ဓာ၌ ဥဒယရှုပုံ (ပထမနည်း)

**(စေတနာကို ကောက်ပါ။)**

- ၁။ အဝိဇ္ဇာ (၂၀) ဖြစ်ခြင်းကြောင့် (သမ္ပဋိစ္ဆိုင်း) သင်္ခါရ ဖြစ်၏။
- ၂။ တဏှာ (၂၀) ဖြစ်ခြင်းကြောင့် (သမ္ပဋိစ္ဆိုင်း) သင်္ခါရ ဖြစ်၏။
- ၃။ ဥပါဒါန် (၂၀) ဖြစ်ခြင်းကြောင့် (သမ္ပဋိစ္ဆိုင်း) သင်္ခါရ ဖြစ်၏။
- ၄။ သင်္ခါရ (၃၄) ဖြစ်ခြင်းကြောင့် (သမ္ပဋိစ္ဆိုင်း) သင်္ခါရ ဖြစ်၏။
- ၅။ ကံ (= သင်္ခါရ-၃၄-၏ ကမ္မသတ္တိ) ဖြစ်ခြင်းကြောင့် (သမ္ပဋိစ္ဆိုင်း) သင်္ခါရ ဖြစ်၏။
- ၆။ ဝတ္ထု (= ဟဒယဝတ္ထု = ၅၄) ဖြစ်ခြင်းကြောင့် (သမ္ပဋိစ္ဆိုင်း) သင်္ခါရ ဖြစ်၏။
- ၇။ အာရုံ (= ရူပါရုံ) ဖြစ်ခြင်းကြောင့် (သမ္ပဋိစ္ဆိုင်း) သင်္ခါရ ဖြစ်၏။
- ၈။ (က) စက္ခုသမ္ဖဿ (= ၈) ဖြစ်ခြင်းကြောင့် (သမ္ပဋိစ္ဆိုင်း) သင်္ခါရ ဖြစ်၏။
  - ( ခ ) သမ္ပဋိစ္ဆိုင်း မနောသမ္ဖဿ (= ၁၁ - စေတနာ = ၁၀) ဖြစ်ခြင်းကြောင့် (သမ္ပဋိစ္ဆိုင်း) သင်္ခါရ ဖြစ်၏။
