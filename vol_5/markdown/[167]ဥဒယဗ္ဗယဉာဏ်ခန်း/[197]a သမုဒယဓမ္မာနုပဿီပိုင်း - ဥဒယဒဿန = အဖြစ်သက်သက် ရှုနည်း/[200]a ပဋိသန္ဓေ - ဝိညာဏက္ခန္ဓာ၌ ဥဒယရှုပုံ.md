### ပဋိသန္ဓေ - ဝိညာဏက္ခန္ဓာ၌ ဥဒယရှုပုံ

- ၁။ အဝိဇ္ဇာ (၂၀) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) ဝိညာဏ် ဖြစ်၏။
- ၂။ တဏှာ (၂၀) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) ဝိညာဏ် ဖြစ်၏။
- ၃။ ဥပါဒါန် (၂၀) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) ဝိညာဏ် ဖြစ်၏။
- ၄။ သင်္ခါရ (၃၄) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) ဝိညာဏ် ဖြစ်၏။
- ၅။ ကံ (= သင်္ခါရ-၃၄-၏ ကမ္မသတ္တိ) ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) ဝိညာဏ် ဖြစ်၏။
- ၆။ နာမ်-ရုပ် ဖြစ်ခြင်းကြောင့် (ပဋိသန္ဓေ) ဝိညာဏ် ဖြစ်၏။
  - \* (ပဋိသန္ဓေ) ဝိညာဏ်၏ ဖြစ်ခြင်းသဘော။

ဤ၌ နာမ်-ဟူသည်မှာ ပဋိသန္ဓေဝိညာဏ်နှင့်ယှဉ်ဖက် စေတသိက်တရားစုတည်း၊ ရုပ်ဟူသည်မှာ မှီရာဝတ္ထုရုပ်ကိုလည်း ဆိုလို၏၊ အကယ်၍ ပဋိသန္ဓေ၏ အာရုံနိမိတ်သည် ရုပ်တရားဖြစ်ခဲ့သော် ယင်းအာရုံရုပ်တရားကိုလည်း ဆိုလို၏၊ ပဋိသန္ဓေ၏ အာရုံမှာ ရုပ်တရား ဟုတ်-မဟုတ်ကို အသင်သူတော်ကောင်း ကိုယ်တိုင် ဆန်းစစ်ကြည့်ပါလေ။
