### ဝိပဿနာရှုသည့် ဉာဏ်

ဝိပဿနာရှုသည့် ဉာဏ်ဟူသည်မှာ ဝိပဿနာဉာဏ် ပြဓာန်းသည့် မဟာကုသိုလ် (မဟာကြိယာ) မနော-ဒွါရိက ဇောဝီထိ စိတ်အစဉ် နာမ်တရားစုတို့ပင် ဖြစ်ကြောင်းကို အထက်တွင် ရှင်းပြထားသည့်အတိုင်း ဝိပဿနာဉာဏ်နှင့် တကွသော ယင်းနာမ်တရားစုတို့၏ အပျက်ကို မြင်အောင်ကြည့်၍ တစ်ဖန် ထပ်၍ ဝိပဿနာရှုရမည်ဖြစ်သည်။ 
ထိုသို့ ရှုရာ၌ သင်္ခါရတရား = ဉာတတရားတို့၏ အပျက်ကို အာရုံယူ၍ -----

- ၁။ အနိစ္စဟု ရှုသော ဝိပဿနာဉာဏ်ကို 
  - (က) အနိစ္စဟုလည်းကောင်း，
  - ( ခ ) ဒုက္ခဟုလည်းကောင်း，
  - ( ဂ ) အနတ္တဟုလည်းကောင်း，

- ၂။ ဒုက္ခဟု ရှုသော ဝိပဿနာဉာဏ်ကို 
  - (က) အနိစ္စဟုလည်းကောင်း
  - ( ခ ) ဒုက္ခဟုလည်းကောင်း，
  - ( ဂ ) အနတ္တဟုလည်းကောင်း，

- ၃။ အနတ္တဟု ရှုသော ဝိပဿနာဉာဏ်ကို 
  - (က) အနိစ္စဟုလည်းကောင်း，
  - ( ခ ) ဒုက္ခဟုလည်းကောင်း，
  - ( ဂ ) အနတ္တဟုလည်းကောင်း ----

ဤသို့ အသီးအသီး ခွဲ၍ ရှုပါ။ 
ခန္ဓာငါးပါးနည်း， အာယတန တစ်ဆယ့်နှစ်ပါးနည်း， ဓာတ်တစ်ဆယ့်ရှစ်ပါးနည်း， ပဋိစ္စသမုပ္ပါဒ်နည်းစသည်တို့၌လည်း နည်းမှီး၍ ရှုပါလေ။
