### အဟိတ် အကုသလဝိပါက်စိတ် (၇) မျိုး

- ၁။ ဥပေက္ခာသဟဂုတ် စက္ခုဝိညာဏ်စိတ် = ဥပေက္ခာဝေဒနာနှင့် အတူတကွ ဖြစ်သော စက္ခုဝတ္ထုရုပ်ကို မှီဖြစ်၍ ရူပါရုံကို သိသောစိတ် (မြင်သိစိတ်)၊
- ၂။ ဥပေက္ခာသဟဂုတ် သောတဝိညာဏ်စိတ် = ဥပေက္ခာဝေဒနာနှင့် အတူတကွ ဖြစ်သော သောတဝတ္ထုရုပ်ကို မှီဖြစ်၍ သဒ္ဒါရုံကို သိသောစိတ် (ကြားသိစိတ်)၊
- ၃။ ဥပေက္ခာသဟဂုတ် ဃာနဝိညာဏ်စိတ် = ဥပေက္ခာဝေဒနာနှင့် အတူတကွ ဖြစ်သော ဃာနဝတ္ထုရုပ်ကို မှီဖြစ်၍ ဂန္ဓာရုံကို သိသောစိတ် (နံသိစိတ်)၊
- ၄။ ဥပေက္ခာသဟဂုတ် ဇိဝှါဝိညာဏ်စိတ် = ဥပေက္ခာဝေဒနာနှင့် အတူတကွဖြစ်သော ဇိဝှါဝတ္ထုရုပ်ကို မှီဖြစ်၍ ရသာရုံကို သိသောစိတ် (အရသာသိစိတ်)၊
- ၅။ ဒုက္ခသဟဂုတ် ကာယဝိညာဏ်စိတ် = ကိုယ်ဆင်းရဲခြင်း ဒုက္ခဝေဒနာနှင့် အတူတကွ ဖြစ်သော ကာယဝတ္ထုရုပ်ကို မှီဖြစ်၍ ဖောဋ္ဌဗ္ဗာရုံကို သိသောစိတ် (ထိသိစိတ်)၊
- ၆။ ဥပေက္ခာသဟဂုတ် သမ္ပဋိစ္ဆိုင်းစိတ် = ဥပေက္ခာဝေဒနာနှင့် အတူတကွ ဖြစ်သော အာရုံကို လက်ခံသောစိတ်၊
- ၇။ ဥပေက္ခာသဟဂုတ် သန္တီရဏစိတ် = ဥပေက္ခာဝေဒနာနှင့် အတူတကွ ဖြစ်သော အာရုံကို စုံစမ်းသောစိတ်၊ ဤစိတ် (၇)မျိုးတို့ကား အလိုမရှိအပ် မနှစ်သက်အပ် မမြတ်နိုးအပ်သော အနိဋ္ဌာရုံနှင့် တွေ့ကြုံရာဝယ် ဝီထိစိတ်တို့၏ အတွင်းဝယ် ပဉ္စဝိညာဏ်， သမ္ပဋိစ္ဆိုင်း， သန္တီရဏ， တဒါရုံကိစ္စများကို ရွက်ဆောင်လျက် ဖြစ်ပေါ်လာကြသော အကုသိုလ်၏ အကျိုးဖြစ်ကုန်သော အကုသလဝိပါက်စိတ်တို့တည်း။ အနိဋ္ဌာရုံများကို ရှေးအကုသိုလ်ကံကြောင့်သာ တွေ့ကြုံရသဖြင့် အနိဋ္ဌာရုံနှင့် ဆုံခိုက် ဖြစ်ပေါ်လာကြသော ယင်းစိတ်တို့မှာလည်း အကုသလဝိပါက် = အကုသိုလ်၏ အကျိုးတရားတို့သာ ဖြစ်ကြရသည်။
