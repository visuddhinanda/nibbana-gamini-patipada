### အာဒိကမ္မိက ပထမဈာနဝီထိပုံစံ

မန္ဒပုဂ္ဂိုလ် = န - ဒ - မ - ပ - ဥ - နု - ဂေါ - ဈာန် - ဘဝင် ...

တိက္ခပုဂ္ဂိုလ် = န - ဒ - မ - ဥ - နု - ဂေါ - ဈာန် - ဘဝင် ...

ဒုတိယဈာန် တတိယဈာန် စတုတ္ထဈာန်တို့၌လည်း နည်းတူချလေ။

**န = ဘဝင်္ဂစလန** = လှုပ်သောဘဝင် (အတိတ် မရဏာသန္နဇော၏ အာရုံ)၊

**ဒ = ဘဝင်္ဂုပစ္ဆေဒ** = ရပ်သောဘဝင် (အတိတ် မရဏာသန္နဇော၏ အာရုံ)၊

**မ = မနောဒွါရာဝဇ္ဇန်း** = ပဋိဘာဂနိမိတ်ကို ဆင်ခြင်သောစိတ်၊

**ပ = ပရိကံ** = အပ္ပနာဈာန်ဖြစ်ပေါ်လာရန် ပြုပြင် စီရင်တတ်သောစိတ်၊

**ဥ = ဥပစာရ** = အပ္ပနာဈာန်၏ အနီး၌ ကပ်၍ ဖြစ်တတ်သောစိတ်၊

**နု = အနုလောမ** = အနုလုံ = ရှေးရှေး ပရိကံနှင့် နောက်နောက် အပ္ပနာတို့အား လျော်သောစိတ်၊

**ဂေါ = ဂေါတြဘု** = ကာမအနွယ်ကို လွှမ်းမိုး ဖိစီးတတ် ဖြတ်တတ်သောစိတ်၊

**ဈာန်** = ပဋိဘာဂနိမိတ်အာရုံကို စိုက်စိုက်စူးစူး အထူးမြဲမြံ ရှုအားသန်သောစိတ်၊

(အာရမ္မဏူပနိဇ္ဈာနဈာန် = ပဋိဘာဂနိမိတ်အာရုံသို့ ကပ်၍ စိုက်စိုက်စူးစူးအထူးမြဲမြံ ရှုအားသန်သောစိတ် ---- ဟူလိုသည်။)
