### လောဘ-ဒိဋ္ဌိ အုပ်စု နာမ်တရား (၂၀)

- ၁။ **အသိစိတ်** = စက္ခုအကြည်ဓာတ် အာရုံကို ရယူခြင်းသဘော။
- ၂။ **ဖဿ** = စက္ခုအကြည်ဓာတ် အာရုံကို တွေ့ထိသော အခြင်းအရာအားဖြင့် ဖြစ်ခြင်းသဘော။ = စက္ခုအကြည်ဓာတ် အာရုံနှင့် အသိစိတ်ကို ဆက်စပ်ပေးခြင်းသဘော။ (ရသ)
- ၃။ **ဝေဒနာ** = စက္ခုအကြည်ဓာတ် အာရုံ၏ အရသာကို ဝမ်းမြောက်စွာ ခံစားခြင်းသဘော။ (ဤ၌ သောမနဿဝေဒနာတည်း။ ဥပေက္ခာဖြစ်လျှင် အလယ်အလတ် ခံစား၏။)
- ၄။ **သညာ** = စက္ခုအကြည်ဓာတ် အာရုံကို (သုဘ)ဟု မှတ်သားခြင်းသဘော။
- ၅။ **စေတနာ** = စက္ခုအကြည်ဓာတ် အာရုံပေါ်သို့ ယှဉ်ဖက် စိတ်-စေတသိက် = သမ္ပယုတ်တရားစုတို့ကို ရောက်အောင် စေ့ဆော်ပေးခြင်း = နှိုးဆော်ပေးခြင်းသဘော။
- ၆။ **ဧကဂ္ဂတာ** = စက္ခုအကြည်ဓာတ် အာရုံတစ်ခုတည်းပေါ်သို့ စိတ်ကျရောက်နေခြင်းသဘော။ = စက္ခုအကြည်ဓာတ် အာရုံ၌ သမ္ပယုတ်တရားတို့ကို ဖရိုဖရဲ မကျဲစေခြင်းသဘော။
- ၇။ **ဇီဝိတ** = စက္ခုအကြည်ဓာတ် အာရုံ၌ ယှဉ်ဖက် စိတ်-စေတသိက် သမ္ပယုတ်တရားတို့ကို စောင့် ရှောက်ပေးခြင်းသဘော။ (နာမ်သက်စောင့်ဓာတ်)
- ၈။ **မနသိကာရ** = စက္ခုအကြည်ဓာတ် အာရုံဘက်သို့ စိတ်ကို (= သမ္ပယုတ်တရားတို့ကို) ဦးလှည့် တွန်းပို့ပေးခြင်းသဘော။ = စက္ခုအကြည်ဓာတ် အာရုံကို နှလုံးသွင်းခြင်းသဘော။
- ၉။ **ဝိတက်** = စက္ခုအကြည်ဓာတ် အာရုံပေါ်သို့ စိတ်ကို (= သမ္ပယုတ်တရားတို့ကို) ရှေးရှူတင်ပေးခြင်းသဘော (= ကြံစည်ခြင်းသဘော။)
- ၁၀။ **ဝိစာရ** = စက္ခုအကြည်ဓာတ်အာရုံကို ထပ်ကာထပ်ကာ သုံးသပ်ဆင်ခြင်ခြင်းသဘော ဆုပ်နယ်ခြင်းသဘော = အာရုံ ထပ်၍ ထပ်၍ ယူခြင်းသဘော။
- ၁၁။ **အဓိမောက္ခ** = စက္ခုအကြည်ဓာတ် အာရုံကို (သုဘဟု) ဆုံးဖြတ်ချက် ချခြင်းသဘော။
- ၁၂။ **ဝီရိယ** = စက္ခုအကြည်ဓာတ် အာရုံပေါ်၌ လောဘ-ဒိဋ္ဌိ စသော အကုသိုလ်တရားများ ဖြစ်အောင်(= ရာဂ ဖြစ်အောင်) ကြိုးစားအားထုတ်ခြင်းသဘော။
- ၁၃။ **ပီတိ** = စက္ခုအကြည်ဓာတ် အာရုံကို နှစ်သက်ခြင်းသဘော။
- ၁၄။ **ဆန္ဒ** = စက္ခုအကြည်ဓာတ် အာရုံကို လိုလားတောင့်တခြင်းသဘော၊ (= ရာဂ ဖြစ်လိုသည့်သဘော။)
- ၁၅။ **မောဟ** = စက္ခုအကြည်ဓာတ် အာရုံ၏ (အသုဘ) သဘောမှန်ကို မသိခြင်းသဘော။ = စက္ခုအကြည်ဓာတ် အာရုံကို (သုဘ)ဟု အသိမှားခြင်းသဘော။
- ၁၆။ **အဟိရိက** = စက္ခုအကြည်ဓာတ် အာရုံပေါ်၌ လောဘ-ဒိဋ္ဌိ စသော အကုသိုလ်ဖြစ်မှုကို မရှက်ခြင်းသဘော။
- ၁၇။ **အနောတ္တပ္ပ** = စက္ခုအကြည်ဓာတ် အာရုံပေါ်၌ လောဘ-ဒိဋ္ဌိ စသော အကုသိုလ်ဖြစ်မှုကို မကြောက်ခြင်းသဘော။
- ၁၈။ **ဥဒ္ဓစ္စ** = စက္ခုအကြည်ဓာတ် အာရုံပေါ်၌ စိတ်မငြိမ်မသက် ပျံ့လွင့်နေခြင်းသဘော။
- ၁၉။ **လောဘ** = စက္ခုအကြည်ဓာတ် အာရုံကို “ငါ့ဟာ”ဟု စွဲယူခြင်းသဘော။ = စက္ခုအကြည်ဓာတ် အာရုံကို လိုလားတပ်မက်ခြင်းသဘော = ကပ်ငြိခြင်းသဘော။
- ၂၀။ **ဒိဋ္ဌိ** = စက္ခုအကြည်ဓာတ် အာရုံကို သုဘဟု စွဲလမ်းယုံကြည်ခြင်းသဘော။ = စက္ခုအကြည်ဓာတ် အာရုံကို သုဘဟု အယူမှားခြင်း = ခံယူချက်မှားခြင်းသဘော။

[**မှတ်ချက်** --- စက္ခုအကြည်ဓာတ်အာရုံကို နိစ္စဟု သုခဟု အတ္တဟု စွဲယူရာ၌လည်း နည်းတူ သဘောပေါက်ပါလေ။ 
နိစ္စဟု သုခဟု အတ္တဟု အသိမှားမှုသဘော အနိစ္စ ဒုက္ခ အနတ္တ သဘောမှန်ကို မသိမှုသဘောကား မောဟ။ 
နိစ္စဟု သုခဟု အတ္တဟု ယုံကြည်ချက်မှားမှု ခံယူချက်မှားမှုကား ဒိဋ္ဌိတည်း။ 
ဝေဒနာမှာ သောမနဿဖြစ်လျှင် ပီတိ ပါ၏။ 
ဥပေက္ခာဖြစ်လျှင် ပီတိ မပါ။ 
သသင်္ခါရိက ဖြစ်၍ တိုက်တွန်းမှုရှိလျှင် ထိန-မိဒ္ဓ ပါ၏၊ အသင်္ခါရိက ဖြစ်၍ တိုက်တွန်းမှု မရှိလျှင် ထိန-မိဒ္ဓ မပါ။ 
(၄) ချက်စီပင် ရှုပါ။ 
အကယ်၍ စက္ခုအကြည်ဓာတ်အာရုံကို ရှုရာ၌ ပီတိ မဖြစ်လျှင် ပီတိ နှုတ်သော် နာမ်တရား (၁၉) ဖြစ်သည်၊ ထိုအခါ ဥပေက္ခာ ဝေဒနာနှင့် ယှဉ်၏။ 
သောမနဿနှင့် မယှဉ်။ 
အကယ်၍ စက္ခုအကြည်ဓာတ်အာရုံ၌ လောဘ ဒိဋ္ဌိ ဖြစ်မှုနှင့်စပ်၍ စိတ်ဓာတ်သည် ထက်သန်မှု သိပ်မရှိလျှင် သသင်္ခါရိက ဖြစ်နေ၍ ထိန-မိဒ္ဓ ယှဉ်မည် ဖြစ်သည်။ 
ထိန-မိဒ္ဓယှဉ်သော နာမ်တရားစုတို့မှာ (၂၂)နှင့် (၂၁) အသီးအသီးဖြစ်သည်။ 
တစ်လုံးစီ ခွဲရှုခြင်း， နာမ်တရားအားလုံးကို ဝီထိစဉ်အတိုင်း ခြုံ၍ရှုခြင်းတို့ဖြင့် ကြိမ်ဖန်များစွာ လေ့ကျင့်ပါ။]
