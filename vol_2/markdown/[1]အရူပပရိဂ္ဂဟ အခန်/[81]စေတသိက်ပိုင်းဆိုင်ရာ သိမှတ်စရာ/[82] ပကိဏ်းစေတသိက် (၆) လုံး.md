### ပကိဏ်းစေတသိက် (၆) လုံး
### (= ပြိုးပြွမ်းယှဉ်သော စေတသိက် ၆-လုံး)

ဤ စေတသိက် (၆)လုံးတို့သည်ကား အချို့သော စိတ်၌သာ ယှဉ်၍ အချို့သော စိတ်၌ မယှဉ်ကြပေ၊ ဟိုတချို့ သည်တချို့ ရောရောပြွမ်းပြွမ်း ယှဉ်သော စေတသိက်များ ဖြစ်ကြသည်။

- ၁။ **ဝိတက်** = အာရုံပေါ်သို့ ယှဉ်ဖက် စိတ်+စေတသိက် (= သမ္ပယုတ်တရားတို့ကို) ရှေးရှုတင်ပေးခြင်းသဘော၊= ကြံစည်ခြင်းသဘော၊
- ၂။ **ဝိစာရ** = အာရုံကို ထပ်ကာ ထပ်ကာ သုံးသပ်ဆင်ခြင်ခြင်း ဆုပ်နယ်ခြင်းသဘော၊= ထပ်၍ ထပ်၍ ယူခြင်းသဘော၊
- ၃။ **အဓိမောက္ခ** = အာရုံကို ဆုံးဖြတ်ချက်ချခြင်းသဘော၊
- ၄။ **ဝီရိယ** = ကြိုးစားအားထုတ်ခြင်းသဘော၊

(အာရုံတစ်ခုပေါ်၌ ယှဉ်ဖက်စိတ်+စေတသိက် = သမ္ပယုတ်တရားတို့ ဖြစ်ပေါ်လာအောင် ကြိုးစားအားထုတ်ခြင်းသဘော ဖြစ်သည်။ 
မကောင်းဘက်၌လည်း လောဘဖြစ်အောင် ဒေါသဖြစ်အောင် မောဟဖြစ်အောင်စသည်ဖြင့် ကြိုးစားမှု， အကောင်းဘက်၌လည်း အလောဘဖြစ်အောင် အဒေါသဖြစ်အောင် အမောဟဖြစ်အောင်စသည်ဖြင့် ကြိုးစားမှုမျိုးတည်း။)

- ၅။ **ပီတိ** = အာရုံကို နှစ်သက်ခြင်းသဘော၊
- ၆။ **ဆန္ဒ** = အာရုံကို လိုလားတောင့်တခြင်းသဘော၊

= အာရုံကို ပြုလုပ်လိုခြင်းသဘော၊

[အာရုံတစ်ခုပေါ်၌ လောဘ ဖြစ်လိုသည့်သဘော， ဒေါသ ဖြစ်လိုသည့်သဘော， မောဟ ဖြစ်လိုသည့်သဘော， အလောဘ ဖြစ်လိုသည့်သဘော， အဒေါသ ဖြစ်လိုသည့်သဘော， အမောဟ ဖြစ်လိုသည့်သဘော --ဤသို့ စသည်ဖြင့် သိပါလေ။]

အထက်တွင် ဖော်ပြခဲ့သော သဗ္ဗစိတ္တသာဓာရဏစေတသိက် (၇)လုံးနှင့် ဤပကိဏ်းစေတသိက် (၆)လုံး= နှစ်ရပ်ပေါင်း (၁၃)လုံးကုန်သော စေတသိက်တို့ကို **အညသမာန်း**စေတသိက်ဟု ခေါ်၏။ 
ယင်း စေတသိက်တို့သည် အကုသိုလ်စိတ်စေတသိက်တို့နှင့် ယှဉ်ခိုက် အကုသိုလ်ဟူသော အမည်ကို ရရှိကြ၏။ 
ကုသိုလ်စိတ် စေတသိက်တို့နှင့် ယှဉ်ခိုက် ကုသိုလ်ဟူသော အမည်ကို ရရှိကြ၏၊ ဝိပါက်+ကြိယာ = အဗျာကတတရားတို့နှင့် ယှဉ်ခိုက် အဗျာကတဟူသော အမည်ကို ရရှိကြ၏။ 
တစ်နည်းဆိုရသော် အသောဘဏစိတ် = မတင့်တယ်သောစိတ်တို့နှင့် ယှဉ်ခိုက် အသောဘဏဟူသော အမည်ကို ရရှိကြ၏၊ သောဘဏစိတ် = တင့်တယ်သောစိတ်တို့နှင့်ယှဉ်ခိုက် သောဘဏဟူသော အမည်ကို ရရှိကြ၏။ 
အကောင်းအဆိုး နှစ်ဖက်ရ စေတသိက်တို့ ဖြစ်ကြသည်။
