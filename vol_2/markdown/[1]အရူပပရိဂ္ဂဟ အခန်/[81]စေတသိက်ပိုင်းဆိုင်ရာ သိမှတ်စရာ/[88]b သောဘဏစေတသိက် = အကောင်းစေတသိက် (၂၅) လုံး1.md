### သောဘဏစေတသိက် = အကောင်းစေတသိက် (၂၅) လုံး

သောဘဏစေတသိက် (၂၅)လုံးမှာ ----

- (က) သောဘဏသာဓာရဏစေတသိက် = သောဘဏအမည်ရသော အကောင်းစိတ်အားလုံးနှင့် ဆက်ဆံသောစေတသိက် ---- (၁၉) လုံး，
- ( ခ ) ဝိရတီစေတသိက် ---- (၃) လုံး，
- ( ဂ ) အပ္ပမညာစေတသိက် ---- (၂) လုံး，
- (ဃ) ပညိန္ဒြေစေတသိက် ---- (၁) လုံး ---<br>အားလုံးပေါင်းသော် ---- (၂၅) လုံး ဖြစ်သည်။
