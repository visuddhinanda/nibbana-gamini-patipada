### ရှုကွက် အတိုချုပ်

- ၁။ **ဒိဋ္ဌ** ---- (က) မြင်အပ်သောတရား = ရူပါရုံ
  - ( ခ ) မြင်တတ်သောတရား = စက္ခုဒွါရဝီထိ မနောဒွါရဝီထိ နာမ်တရားစု
- ၂။ **သုတ** ---- (က) ကြားအပ်သောတရား = သဒ္ဒါရုံ
  - ( ခ ) ကြားတတ်သောတရား = သောတဒွါရဝီထိ မနောဒွါရဝီထိ နာမ်တရားစု
- ၃။ **မုတ** ---- (က) တွေ့အပ်သောတရား ---- ၁။ ဂန္ဓာရုံ
- ၂။ ရသာရုံ
- ၃။ ဖောဋ္ဌဗ္ဗာရုံ
  - ( ခ ) တွေ့တတ်သောတရား ---- 

- ၁။ ဃာနဒွါရဝီထိ ---- မနောဒွါရဝီထိ နာမ်တရားစု
- ၂။ ဇိဝှါဒွါရဝီထိ ---- မနောဒွါရဝီထိ နာမ်တရားစု
- ၃။ ကာယဒွါရဝီထိ ---- မနောဒွါရဝီထိ နာမ်တရားစု
- ၄။ **ဝိညာတ** ---- (က) သိအပ်သောတရား = ဓမ္မာရုံ
  - ( ခ ) သိတတ်သောတရား = မနောဒွါရဝီထိနာမ်တရားစု

ဤဖော်ပြပါ (၄)မျိုးသော ---- ဒိဋ္ဌ-သုတ-မုတ-ဝိညာတ တရားတို့ကို ရုပ်တုံးရုပ်ခဲ နာမ်တုံးနာမ်ခဲဟူသောရုပ်ဃန နာမ်ဃနတို့ကို ဉာဏ်ဖြင့် ဖြိုခွဲဖျက်ဆီးလျက် ပရမတ်သို့ ဉာဏ်အမြင်ဆိုက်အောင် ရှေးဦးစွာ သိမ်းဆည်းပါ။ 
ဉာဏ်ဖြင့် ပိုင်းခြားယူပါ။ (သိမ်းဆည်းပုံ အကျယ်ကို ဖော်ပြခဲ့ပြီး ဖြစ်သည်။)

ထိုနောင် ယင်းရုပ်နာမ်တို့၏ အကြောင်းတရားများကို ရှာဖွေလျက် အကြောင်းတရား အကျိုးတရားတို့ကိုသိမ်းဆည်းပါ။ 
(ပဋိစ္စသမုပ္ပါဒ်ပိုင်း ပစ္စယပရိဂ္ဂဟဉာဏ်ပိုင်းတွင် ထင်ရှားလတ္တံ့။)

ထိုနောင် အကြောင်းတရားနှင့် တကွသော ယင်းရုပ်နာမ်တို့ကို လက္ခဏာရေးသုံးတန်တင်ကာ ဝိပဿနာရှုပွားသုံးသပ်ပါ။ (ဝိပဿနာပိုင်းတွင် ထင်ရှားလတ္တံ့။) 
ထိုနောင် ရှေ့သို့ ဆက်လက်ကျင့်သုံးရမည့် လုပ်ငန်းခွင်ကို ဗာဟိယအား ဘုရားရှင်က **ယတော ခေါ တေ ဗာဟိယ** ---- စသည်ဖြင့် ဆက်လက် ဟောကြားတော်မူ၏။
