### ဒုက္ခသမုဒယ အရိယသစ္စာ

ကတမဉ္စ ဘိက္ခဝေ ဒုက္ခသမုဒယံ အရိယသစ္စံ။ 
ယာယံ တဏှာ ပေါနောဗ္ဘဝိကာ နန္ဒီရာဂသဟဂတာ တတြတတြာဘိနန္ဒိနီ။ 
သေယျထိဒံ၊ ကာမတဏှာ ဘဝတဏှာ ဝိဘဝတဏှာ။ (အဘိ၊၂၊၁၀၆။ မ၊၁၊၈၄။)

= ရဟန်းတို့ ...ဒုက္ခ အရိယသစ္စာဖြစ်ကြောင်း ဒုက္ခသမုဒယ အရိယသစ္စာတရားဟူသည် အဘယ်နည်း？တစ်ဖန် ဘဝသစ်ကို ဖြစ်စေတတ်သော နှစ်သက်တပ်မက်ခြင်း = နန္ဒီရာဂနှင့် အတူတကွ ဖြစ်သော ထိုထိုဘဝ，ထိုထိုအာရုံ၌ အလွန်နှစ်သက်ခြင်းရှိသော တဏှာတည်း။ 
ထိုတဏှာသည် အဘယ်နည်း？ ---- ကာမတဏှာ，ဘဝ တဏှာ， ဝိဘဝတဏှာတို့တည်း။ (အဘိ၊၂၊၁၀၆။ မ၊၁၊၈၄။)

**ကာမတဏှာ**တိ ကာမေ တဏှာ ကာမတဏှာ။ 
ပဉ္စကာမဂုဏိကရာဂဿေတံ အဓိဝစနံ။ 
ဘဝေ တဏှာ**ဘဝတဏှာ။** 
ဘဝပတ္ထနာဝသေန ဥပ္ပန္နဿ သဿတဒိဋ္ဌိသဟဂတဿ ရူပါရူပဘဝရာဂဿ စ ဈာနနိကန္တိယာစေတံ အဓိဝစနံ။ 
ဝိဘဝေ တဏှာ **ဝိဘဝတဏှာ။** 
ဥစ္ဆေဒဒိဋ္ဌိသဟဂတဿ ရာဂဿေတံ အဓိဝစနံ။ (အဘိ၊ဋ္ဌ၊၂၊၁၀၄။)

ရူပါရူပဘဝရာဂေါ ဝိသုံ ဝက္ခတီတိ ကာမဘဝေ ဧဝ ဘဝပတ္ထနာဥပ္ပတ္တိ ဝုတ္တာတိ ဝေဒိတဗ္ဗာ။ (မူလဋီ၊၂၊၆၉။)

**၁။ ကာမတဏှာ** ---- ငါးပါးအာရုံ ကာမဂုဏ်၌ တွယ်တာတပ်မက်သော တဏှာသည် ကာမတဏှာ မည်၏။ 
ဤ ကာမတဏှာဟူသော အမည်သည် အာရုံငါးပါး ကာမဂုဏ်တရား၌ တွယ်တာ တပ်မက်တတ်သော ရာဂ၏အမည် ဖြစ်သည်။

**၂။ ဘဝတဏှာ ----**
- (က) ကာမဘဝကို လိုလားတောင့်တသည်၏ အစွမ်းဖြင့် ဖြစ်ပေါ်လာသော ကာမဘဝ၌ တွယ်တာ တပ်မက်သော တဏှာသည်လည်း ဘဝတဏှာ မည်၏။
- ( ခ ) အတ္တသည် မပျက်မစဲ အမြဲဖြစ်၏ဟူသော သဿတဒိဋ္ဌိ အယူကို ယူပြီးလျှင် ထိုအယူကို နှစ်သက်သော တဏှာသည်လည်း ဘဝတဏှာ မည်၏။
- ( ဂ ) ရူပဘဝ အရူပဘဝကို တွယ်တာ တပ်မက်သော ရူပရာဂ အရူပရာဂဟု ခေါ်ဆိုအပ်သော တဏှာသည်လည်း ဘဝတဏှာ မည်၏။
- (ဃ) ရူပဘဝ အရူပဘဝ၏ ဖြစ်ကြောင်း ဖြစ်သော ဈာန်တရား၌ တွယ်တာတပ်မက်တတ်သော ဈာနနိကန္တိဟု ခေါ်ဆိုအပ်သော တဏှာသည်လည်း ဘဝတဏှာ မည်၏။

**၃။ ဝိဘဝ တဏှာ ----** အတ္တသည် မမြဲ ပြတ်စဲ၏၊ အတ္တသည် သေလျှင် ပြတ်၏။ ---- 
ဟု ယူဆအပ်သော အတ္တဝါဒသည် ဝိဘဝ မည်၏၊ ထို ဝိဘဝ အမည်ရသော အတ္တဝါဒကို စွဲယူသော ဒိဋ္ဌိသည် ဥစ္ဆေဒဒိဋ္ဌိ = ဝိဘဝဒိဋ္ဌိမည်၏။ 
ထို ဥစ္ဆေဒဒိဋ္ဌိ = ဝိဘဝဒိဋ္ဌိကို တွယ်တာ တပ်မက်သော ဥစ္ဆေဒဒိဋ္ဌိ = ဝိဘဝဒိဋ္ဌိနှင့် အတူတကွ ဖြစ်သောရာဂသည် ဝိဘဝတဏှာ မည်၏။ (အဘိ၊ဋ္ဌ၊၂၊၁၀၄။ မူလဋီ၊၂၊၆၉။)

ဤကား သုတ္တန္တဘာဇနီယနည်းတည်း။ 
အဘိဓမ္မဘာဇနီယနည်းအားဖြင့် သမုဒယသစ္စာကို အောက်ပါအတိုင်း ငါးမျိုးခွဲ၍ ဟောကြားထားတော်မူ၏။
