### ဥပစာရဈာန်

တဿေဝံ ဝါယမမာနဿ နစိရေနေ၀ ဓာတုပ္ပဘေဒါ၀ဘာသနပညာပရိဂ္ဂဟိတော သဘာ၀ဓမ္မာရမ္မဏ တ္တာ အပ္ပနံ အပ္ပတ္တော ဥပစာရမတ္တော သမာဓိ ဥပ္ပဇ္ဇတိ။ ဝိသုဒ္ဓိ၊၁၊၃၄၇။)

ဧဝံ ပန ဝီရိယသမတံ ယောဇေတွာ တသ္မိံ ယေ၀ လက္ခဏေ မနသိကာရံ ပ၀တ္တေန္တဿ ယဒါ သဒ္ဓါဒီနိ ဣန္ဒြိယာနိ လဒ္ဓသမတာနိ သုဝိသဒါနိ ပဝတ္တန္တိ၊ တဒါ အသဒ္ဓိယာဒီနံ ဒူရီဘာဝေန သာတိသယံ ထာမပ္ပတ္တေဟိ သတ္တဟိ ဗလေဟိ လဒ္ဓူပတ္ထမ္ဘာနိ ဝိတက္ကာဒီနိ ဈာနင်္ဂါနိ ပဋုတရာနိ ဟုတွာ ပါတုဘဝန္တိ၊ တေသံ ဥဇုဝိပစ္စ နီကတာယ နီ၀ရဏာနိ ဝိက္ခမ္ဘိတာနိယေ၀ ဟောန္တိ သဒ္ဓိံ တဒေကဋ္ဌေဟိ ပါပဓမ္မေဟိ။ 
ဧတ္တာ၀တာ စာနေန ဥပစာရဇ္ဈာနံ သမဓိဂတံ ဟောတိ ဓာတုလက္ခဏာရမ္မဏံ။ 
တေန ဝုတ္တံ **“ပုနပ္ပုနံ ပထဝီဓာတု အာပေါဓာတူတိ။ ပ** **။ ဥပစာရမတ္တော သမာဓိ ဥပ္ပဇ္ဇတီ”**တိ။ (မဟာဋီ၊၁၊၄၃၅။)

အထက်ပါ အဋ္ဌကထာ ဋီကာတို့၏ ဆိုလိုရင်း သဘောတရားမှာ ဤသို့ ဖြစ်၏။

ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာကို ဟုတ်တိုင်းမှန်စွာ သိအောင် ကြိုးစားအားထုတ်ခြင်း ဝီရိယ， ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာ အာရုံပေါ်၌ ဘာ၀နာစိတ်ကို ငြိမ်ဝပ်စွာ ကပ်၍ တည်နေအောင် ကြိုးစားအားထုတ်ခြင်း ဝီရိယနှင့် ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံ၌ ဘာ၀နာစိတ်၏ ငြိမ်ဝပ်စွာ ကပ်၍ တည်နေမှုဟူသော သမာဓိသည် ရှိ၏။ 
ဤဝီရိယနှင့် သမာဓိ နှစ်ပါးကို ညီမျှအောင် ယှဉ်စေ ဖြစ်စေ၍ ထို ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံ၌ပင်လျှင် နှလုံးသွင်းမှု မနသိကာရကို ကြိမ်ဖန်များစွာ ဖြစ်စေတတ်သော ယောဂါ၀စရ ရဟန်းတော်၏ သန္တာန်ဝယ် အကြင်အခါ၌ ---- 

- ၁။ ဓာတ်ကြီး (၄)ပါးကို အာရုံယူ၍ သမာဓိထူထောင်နေသော သမာဓိသိက္ခာ = သမာဓိကျင့်စဉ်နှင့် ဓာတ်ကြီး (၄)ပါးတို့၏ ကိုယ်ပိုင်သဘာ၀လက္ခဏာကို ထိုးထွင်းသိနေသော ပညာသိက္ခာ = ပညာကျင့်စဉ်ဖြစ်သော ဓာတ်ကမ္မဋ္ဌာန်းဘာ၀နာ လုပ်ငန်းရပ်၌ ယုံကြည်မှု သဒ္ဓါတရားတည်းဟူသော သဒ္ဓိန္ဒြေ， 
- ၂။ ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံကို သိအောင် ကြိုးစားအားထုတ်မှု ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံ၌ ဘာ၀နာစိတ်ကို ငြိမ်ဝပ်စွာ ကပ်၍ တည်နေအောင် ကြိုးစားအားထုတ်မှု ဝီရိယတရားတည်းဟူသော ဝီရိယိန္ဒြေ， 
- ၃။ ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံကိုသာ ထပ်ကာထပ်ကာ သိမ်းဆည်းခြင်း အမှတ်ရခြင်း သတိတရားတည်းဟူသော သတိန္ဒြေ，
- ၄။ ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံပေါ်၌ စိတ်တည်ငြိမ်နေမှု သမာဓိတရားတည်းဟူသော သမာဓိန္ဒြေ，
- ၅။ ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာ အသီးအသီးကို ခွဲခြားစိတ်ဖြာ၍ ထွင်းဖောက် သိမြင်ခြင်း ပညာတရားတည်းဟူသော ပညိန္ဒြေ ----

ဤဣန္ဒြေ (၅)ပါးတို့သည် တစ်ပါးကို တစ်ပါး မလွန်ဘဲ ရအပ်သော ညီမျှမှု ရှိကုန်၏၊ ကောင်းစွာ သန့် ရှင်းကုန်၏၊ ထိုအခါ၌ မယုံကြည်ခြင်း， ပျင်းရိခြင်း， သတိမေ့လျော့ခြင်း， စိတ်ပျံ့လွင့်ခြင်း， ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာကို ဟုတ်တိုင်းမှန်စွာ ထိုးထွင်းမသိခြင်း တွေဝေခြင်းစသော ဆန့်ကျင်ဘက်တရားတို့၏ ဝေးကွာသည်၏ အဖြစ်ကြောင့် အလွန်အကဲ အားကောင်းခြင်းသို့ ရောက်ကုန်သော အားအစွမ်းနှင့် ပြည့်စုံ ကုန်သော ----

- ၁။ ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာကို အောက်မေ့ အမှတ်ရနေသော အဖန်ဖန် အထပ်ထပ် သိမ်း ဆည်းနေသော သတိသမ္ဗောဇ္ဈင်，
- ၂။ ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာကို စူးစမ်းဆင်ခြင် သုံးသပ်နေသော ထိုးထိုးထွင်းထွင်း သိနေသော ဓမ္မဝိစယသမ္ဗောဇ္ဈင်，
- ၃။ ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံ၌ ဓမ္မဝိစယဖြစ်အောင် (သမ္ပယုတ်တရားစုတို့ ဖြစ်အောင်) အထူးသဖြင့် ငြိမ်ဝပ်စွာ ကပ်၍တည်နေမှု သမာဓိဖြစ်အောင် ကြိုးစားအားထုတ်နေသော ဝီရိယသမ္ဗောဇ္ဈင်， 
 - ၄။ ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံ၌ ဘာ၀နာစိတ်၏ ကြည်ရွှင် ဝမ်းမြောက်ခြင်း ပီတိ သမ္ဗောဇ္ဈင်，
- ၅။ ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံ၌ ဘာ၀နာစိတ်စေတသိက်တို့၏ ငြိမ်းအေးနေခြင်း ပဿဒ္ဓိသမ္ဗောဇ္ဈင်，
- ၆။ ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံ၌ ဘာ၀နာစိတ်၏ ငြိမ်ဝပ်စွာ ကပ်၍ တည်နေမှုသဘော သမာဓိသမ္ဗောဇ္ဈင်，
- ၇။ ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံ၌ ဘာ၀နာစိတ်ကို တက်ကြွခြင်း တွန့်ဆုတ်ခြင်းဘက်သို့ မရောက်အောင် အညီအမျှထားခြင်း သဘောဟူသော ဥပေက္ခာသမ္ဗောဇ္ဈင် ----

ဤ သတိ-ဓမ္မဝိစယ-ဝီရိယ-ပီတိ-ပဿဒ္ဓိ-သမာဓိ-ဥပေက္ခာ ဟူသော ဗောဇ္ဈင် (၇)ပါး ခွန်အား စွမ်းအား ကြီးတို့သည် အားပေးထောက်ပံ့အပ်ကုန်သည် ဖြစ်၍， ထိုခွန်အားကြီး စွမ်းအားကြီးတို့၏ အားပေးထောက်ပံ့မှု ကို ရကုန်သည် ဖြစ်၍ ----

- ၁။ ဝိတက် = ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံပေါ်သို့ စိတ်ကို ရှေးရှုတင်ပေးခြင်းသဘော，
- ၂။ ဝိစာရ = ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာ အာရုံကို ထပ်ကာထပ်ကာ သုံးသပ်ဆင်ခြင်ခြင်းသဘော，
- ၃။ ပီတိ = ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံကို နှစ်သက်ခြင်းသဘော，
- ၄။ သုခ = ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံ၏ အရသာကို ချမ်းချမ်းသာသာ ခံစားခြင်းသဘော，
- ၅။ ဧကဂ္ဂတာ = ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာအာရုံ တစ်ခုတည်းပေါ်သို့ စိတ်ကျရောက် တည် ငြိမ်နေခြင်းသဘော - ဟူသော

ဤဈာန်အင်္ဂါ (၅)ပါးတို့သည် အထူးထက်မြက်ကုန်သည် ဖြစ်၍ ထင်ရှားလာကုန်၏။

- ၁။ ကာမဂုဏ်တို့၌ လိုလားတောင့်တမှု ကာမရာဂတည်းဟူသော ကာမစ္ဆန္ဒနီ၀ရဏ，
- ၂။ သတ္တဝါသင်္ခါရတို့အပေါ်၌ ဖျက်ဆီးလိုသော ပျက်စီးစေလိုသော စိတ်ခက်ထန် ကြမ်းတမ်းမှု ဒေါသတည်း ဟူသော ဗျာပါဒနီ၀ရဏ，
- ၃။ စိတ်စေတသိက်တို့၏ ထိုင်းမှိုင်းမှုတည်းဟူသော ထိနမိဒ္ဓနီ၀ရဏ，
- ၄။ (က) စိတ်ပျံ့လွင့်မှု ဥဒ္ဓစ္စ， (ခ) ပြုပြီး အကုသိုလ်တရား ဒုစရိုက်တရား， မပြုလိုက်မိသော မကျင့်လိုက် မိသော ကုသိုလ်တရား သုစရိုက်တရားတို့၌ နောင်တတစ်ဖန် ပူပန်မှု ကုက္ကုစ္စတည်းဟူသော ဥဒ္ဓစ္စ ကုက္ကုစ္စနီ၀ရဏ，
- ၅။ ဓာတ်ကမ္မဋ္ဌာန်းလမ်းမှ ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာကို အာရုံယူ၍ ထူထောင်ထားအပ်သော သဒိသူပစာရအားဖြင့် ဥပစာရသမာဓိ အမည်ရသော သမာဓိသိက္ခာ = သမာဓိကျင့်စဉ်နှင့် ဓာတ်ကြီး (၄)ပါးတို့၏ သဘာ၀လက္ခဏာကို ထွင်းဖောက်သိမြင်သော ပညာသိက္ခာ = ပညာကျင့်စဉ်တို့၌ သို့လော သို့လော တွေးတောယုံမှားခြင်း ဝိစိကိစ္ဆာနီ၀ရဏတည်းဟူသော ----

ဘာ၀နာစိတ်အစဉ်ကို ဘာ၀နာလမ်းကြောင်းပေါ်သို့ မရောက်အောင် တားမြစ် ပိတ်ပင်တတ်ကုန်သော ဤနီ၀ရဏတရား (၅)ပါးတို့ကား ဝိတက် ဝိစာရ ပီတိ သုခ ဧကဂ္ဂတာဟူသော ဈာန်အင်္ဂါ (၅)ပါးတို့နှင့် တိုက်ရိုက် ဖြောင့်ဖြောင့် ဆန့်ကျင်ဘက် ဖြစ်ကြကုန်၏။ 
နီ၀ရဏတရား (၅)ပါးတို့သည် ထိုဈာန်အင်္ဂါ (၅)ပါးတို့၏ တိုက်ရိုက် ဖြောင့်ဖြောင့် ဆန့်ကျင်ဘက်ဖြစ်သောကြောင့် ဈာန်အင်္ဂါများသည် အလွန်ထက်မြက်စူးရှ ထင်ရှားလာကြသော အခါ နီ၀ရဏတရားတို့သည် ခွာအပ်သည်သာလျှင် ဖြစ်ကုန်၏၊ ကွာသွားကြကုန်၏။ 
နီ၀ရဏတရားတို့သာမက ထိုနီ၀ရဏတရားတို့နှင့် တူသော တည်ရာရှိ ကြကုန်သော ပြိုင်တူ ယှဉ်တွဲဖြစ်ဖက် အကုသိုလ်တရားတို့ကိုလည်း ခွာထားအပ်သည်သာ ဖြစ်ကုန်၏။ 
ဤမျှသော အတိုင်းအရှည်ဖြင့် ဤယောဂါ၀စရပုဂ္ဂိုလ်သည် ဓာတ်ကြီး (၄)ပါး တို့၏ သဘာ၀လက္ခဏာလျှင် အာရုံရှိသော ဥပစာရဈာန်ကို ကောင်းစွာ ရအပ်သည် ဖြစ်ပေသည်။ ထိုကြောင့် (ဝိသုဒ္ဓိ၊၁၊၃၄၇။) အဋ္ဌကထာ၌ ----

“ကာလအရာ ကာလအထောင် ကာလအသိန်းတိုင်အောင် အကြိမ်ကြိမ် အဖန်ဖန် ပထဝီဓာတ် အာပေါ ဓာတ် တေဇောဓာတ် ဝါယောဓာတ်ဟူ၍ ဓာတ်သဘောမျှအားဖြင့် သတ္တဝါ မဟုတ်သောအားဖြင့် ဇီ၀အတ္တ မဟုတ်သောအားဖြင့် ----

- ၁။ ဆင်ခြင်အပ်၏ = ကောင်းစွာ အဖန်တလဲလဲ နှလုံးသို့ ဆောင်အပ်၏။
- ၂။ နှလုံးသွင်းအပ်၏ = အဖန်တလဲလဲ ပွားစေသောအားဖြင့် ဓာတ်၏ သဘာ၀လက္ခဏာကို နှလုံး၌ ထား အပ်၏။
- ၃။ အသီးအသီး တစ်လုံးစီ တစ်လုံးစီ စူးစူးစိုက်စိုက် ရှုအပ်၏ = ဓာတ်၏ သဘာ၀လက္ခဏာကိုပင် ဘာ၀နာ ဖြင့်ပြီးသော ဉာဏ်စက္ခုဖြင့် အဆင့်ဆင့် ထပ်ကာထပ်ကာ ဆင်ခြင်သောအားဖြင့် ရှုအပ်၏။

ဤဆိုအပ်ခဲ့ပြီးသော အပြားအားဖြင့် အားထုတ်သော ဘာ၀နာမှု၌ ကောင်းစွာ ယှဉ်စပ်တတ်သော ထို ယောဂါ၀စရ ရဟန်းတော်၏ သန္တာန်၌ မကြာမြင့်မီ အချိန်ကာလအတွင်းမှာပင်လျှင် ပထဝီဓာတ် အာပေါဓာတ် တေဇောဓာတ် ဝါယောဓာတ်တို့၏ ကိုယ်ပိုင်သဘာ၀လက္ခဏာ အထူးအပြားကို ကွဲကွဲပြားပြား ထင်စေတတ် လင်းစေတတ်သော ပညာ， ဆီမီးကဲ့သို့ ထွန်းလင်းပြခြင်း ကိစ္စရှိသော ပညာဖြင့် ထက်ဝန်းကျင်မှ သိမ်းဆည်း ထားအပ်သော နက်နဲသော ပရမတ္ထဓမ္မသဘာ၀သတ္တိ သဘာ၀ဓမ္မလျှင် အာရုံရှိသည်၏ အဖြစ်ကြောင့် အပ္ပနာ ခေါ် မဟဂ္ဂုတ်ဈာန်သို့ မရောက်သော = အပ္ပနာသမာဓိအဆင့်သို့ မရောက်သော ဥပစာရမျှသာဖြစ်သော သိခါ ပတ္တ = အထွတ်အထိပ်သို့ ရောက်သော ကာမာ၀စရသမာဓိသည် ဖြစ်ပေါ်လာ၏” - ဟု ဖွင့်ဆိုခြင်း ဖြစ်ပေသည်။ 
ဤကား အထက်ပါ အဋ္ဌကထာ ဋီကာ စကားရပ်တို့၏ ဆိုလိုရင်း ဖြစ်ပါသည်။
