### လက္ခဏ နှင့် ရသ

အထက်တွင် ဖော်ပြခဲ့သော ဝိသုဒ္ဓိမဂ္ဂအဋ္ဌကထာ၏ ဓာတ်ကမ္မဋ္ဌာန်းအကျဉ်းနည်း ပွားများ အားထုတ်ပုံ ဘာ၀နာနယ ပထမနည်း ဒုတိယနည်းတို့တွင် ----

- ၁။ ပထဝီဓာတ်၌ ---- (က) ထဒ္ဓဘာ၀ = မာမှုသဘော，<br>( ခ ) ခရဘာ၀ = ကြမ်းမှုသဘော，
- ၂။ အာပေါဓာတ်၌ ---- (က) ဒြ၀ဘာ၀ = ယိုစီးမှုသဘော，<br> ( ခ ) အာဗန္ဓနဘာ၀ = ဖွဲ့စည်းမှုသဘော，
- ၃။ တေဇောဓာတ်၌ ---- (က) ဥဏှဘာ၀ = ပူမှုသဘော，<br>( ခ ) ပရိပါစနဘာ၀ = ရင့်ကျက်စေမှုသဘော，
- ၄။ ဝါယောဓာတ်၌ ---- (က) ဝိတ္ထမ္ဘနဘာ၀ = ထောက်ကန်မှုသဘော，<br>( ခ ) သမုဒီရဏဘာ၀ = တွန်းကန်မှုသဘော ----

ဤသို့လျှင် ဓာတ်တစ်ပါးတစ်ပါး၌ နှစ်မျိုးစီ ဖော်ပြထား၏။ 
ဤဖော်ပြချက်ကို မဟာဋီကာဆရာတော် ဘုရားက အောက်ပါအတိုင်း ရှင်းလင်းဖွင့်ဆိုထား၏။

- ၁။ (က) ထဒ္ဓဘာ၀ = မာမှုသဘောကား - ပထဝီဓာတ်၏ သဘာ၀လက္ခဏာ，<br>( ခ ) ခရဘာ၀ = ကြမ်းမှုသဘောကား - ဥပဋ္ဌာနာကာရ = ရသကိစ္စ，
- ၂။ (က) ဒြ၀ဘာ၀ = ယိုစီးမှုသဘောကား - အာပေါဓာတ်၏ သဘာ၀လက္ခဏာ<br>( ခ ) အာဗန္ဓနဘာ၀ = ဖွဲ့စည်းမှုသဘောကား - ဥပဋ္ဌာနာကာရ = ရသကိစ္စ
- ၃။ (က) ဥဏှဘာ၀ = ပူမှုသဘောကား - တေဇောဓာတ်၏ သဘာ၀လက္ခဏာ<br>( ခ ) ပရိပါစနဘာ၀ = ရင့်ကျက်စေမှုသဘောကား - ဥပဋ္ဌာနာကာရ = ရသကိစ္စ
- ၄။ (က) ဝိတ္ထမ္ဘနဘာ၀ = ထောက်ကန်မှုသဘောကား - ဝါယောဓာတ်၏ သဘာ၀လက္ခဏာ<br>( ခ ) သမုဒီရဏဘာ၀ = တွန်းကန်မှုသဘောကား - ဥပဋ္ဌာနာကာရ = ရသကိစ္စ

ဤသို့ အသီးအသီး ဖွင့်ဆိုထားပေသည်။ (မဟာဋီ၊၁၊၄၃၃။)
