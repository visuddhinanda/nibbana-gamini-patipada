### မဟာဘုတ် (= ဓာတ်ကြီး) လေးပါး

#### ၁။ ပထဝီဓာတ်

| ဂရုပထဝီအုအုပ်စု | လဟုပထဝီအုပ်စု |
| ---------------- | --------------- |
| ၁။ မာမှု ...သဘော，  | ၂။ ပျော့မှု ...သဘော， |
| ၃။ ကြမ်းမှု ...သဘော， | ၄။ ချောမှု ...သဘော， |
| ၅။ လေးမှု ...သဘော，  | ၆။ ပေါ့မှု ...သဘော， |

ကတမံ တံ ရူပံ ဖောဋ္ဌဗ္ဗာယတနံ။ ပထဝီဓာတု တေဇောဓာတု ဝါယောဓာတု ကက္ခဠံ မုဒုကံ သဏှံ ဖရုသံ ဂရုကံ လဟုကံ ...(အဘိ၊၁၊၁၇၀။ ဓမ္မသင်္ဂဏီပါဠိတော်)

ဧတ္ထ စ “ကက္ခဠံ မုဒုကံ သဏှံ ဖရုသံ (သုခသမ္ဖဿံ ဒုက္ခသမ္ဖဿံ) ဂရုကံ လဟုက”န္တိ ပဒေဟိ ပထဝီဓာတု ဧ၀ ဘာဇိတာ။ (အဘိ၊ဋ္ဌ၊၁၊၃၆၈၊ စာပိုဒ်-၆၄၇။)

- ၁။ **ကက္ခဠ**န္တိ ထဒ္ဓံ = မာ - မှုသဘော，
- ၂။ **မုဒုက**န္တိ အထဒ္ဓံ = ပျော့ - မှုသဘော，
- ၃။ **ဖရုသ**န္တိ ခရံ = ကြမ်း - မှုသဘော，
- ၄။ **သဏှ**န္တိ မဋ္ဌံ = ချော - မှုသဘော，
- ၅။ **ဂရုက**န္တိ ဘာရိယံ = လေး - မှုသဘော，
- ၆။ **လဟုက**န္တိ အဘာရိယံ၊ သလ္လဟုကန္တိ အတ္ထော = ပေါ့ - မှုသဘော， (အဘိ၊ဋ္ဌ၊၁၊၃၆၈။)

#### ၂။ အာပေါဓာတ်

- ၁။ ဒြ၀ဘာ၀ = ပဂ္ဃရဏလက္ခဏာ = ယိုစီးမှုသဘော，
- ၂။ အာဗန္ဓနဘာ၀ = အာဗန္ဓနလက္ခဏာ = ဖွဲ့စည်းမှုသဘော，

#### ၃။ တေဇောဓာတ်

- ၁။ (က) ဥဏှဘာ၀ = ဥဏှတေဇော = ပူမှုသဘော，<br>( ခ ) သီတဘာ၀ = သီတတေဇော = အေးမှုသဘော，
- ၂။ ပရိပါစနဘာ၀ = ရင့်ကျက်စေမှုသဘော，

#### ၄။ ဝါယောဓာတ်

- ၁။ ဝိတ္ထမ္ဘနဘာ၀ = ထောက်ကန်မှုသဘော，
- ၂။ သမုဒီရဏဘာ၀ = တွန်းကန်မှုသဘော，
