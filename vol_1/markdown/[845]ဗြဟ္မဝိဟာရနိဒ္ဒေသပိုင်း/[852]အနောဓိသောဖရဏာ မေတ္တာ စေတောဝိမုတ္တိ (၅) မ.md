### အနောဓိသောဖရဏာ မေတ္တာ စေတောဝိမုတ္တိ (၅) မျိုး

- ၁။ သဗ္ဗေ သတ္တာ (က) အဝေရာ ဟောန္တု၊
  - ( ခ ) အဗျာပဇ္ဇာ ဟောန္တု၊
  - ( ဂ ) အနီဃာ ဟောန္တု၊
  - (ဃ) သုခီ အတ္တာနံ ပရိဟရန္တု။
- ၂။ သဗ္ဗေ ပါဏာ ။ ပ ။
- ၃။ သဗ္ဗေ ဘူတာ ။ ပ ။
- ၄။ သဗ္ဗေ ပုဂ္ဂလာ ။ ပ ။
- ၅။ သဗ္ဗေ အတ္တဘာ၀ပရိယာပန္နာ (က) အဝေရာ ဟောန္တု၊
  - ( ခ ) အဗျာပဇ္ဇာဟောန္တု၊
  - ( ဂ ) အနီဃာ ဟောန္တု၊
  - (ဃ) သုခီ အတ္တာနံ ပရိဟရန္တု။

- ၁။ ခပ်သိမ်းကုန်သော ခန္ဓာငါးပါး၌ ငြိကပ်တွယ်တာခြင်း ရှိကြကုန်သော သတ္တဝါတို့သည် -----
  - (က) ဘေးရန် ကင်းကြပါစေ၊
  - ( ခ ) စိတ်ဆင်းရဲ ကင်းကြပါစေ၊
  - ( ဂ ) ကိုယ်ဆင်းရဲ ကင်းကြပါစေ၊
  - (ဃ) ကျန်းကျန်းမာမာ ချမ်းချမ်းသာသာဖြင့် မိမိတို့၏ ခန္ဓာဝန်ကို ရွက်ဆောင်နိုင်ကြပါစေ။
- ၂။ ခပ်သိမ်းကုန်သော ထွက်သက် ဝင်သက် ရှိကြကုန်သော သတ္တဝါတို့သည်။ ပ ။
- ၃။ ခပ်သိမ်းကုန်သော ကံကိလေသာကြောင့် ထင်ရှားဖြစ်ကြကုန်သော သတ္တဝါတို့သည်။ ပ ။
- ၄။ ခပ်သိမ်းကုန်သော ငရဲသို့ ကျရောက်ခြင်းသဘော ရှိကြကုန်သော သတ္တဝါတို့သည်။ ပ ။
- ၅။ ခပ်သိမ်းကုန်သော ခန္ဓာကိုယ်အတ္တဘော၌ အကျုံးဝင်ကြကုန်သော သတ္တဝါတို့သည်။ ပ ။
