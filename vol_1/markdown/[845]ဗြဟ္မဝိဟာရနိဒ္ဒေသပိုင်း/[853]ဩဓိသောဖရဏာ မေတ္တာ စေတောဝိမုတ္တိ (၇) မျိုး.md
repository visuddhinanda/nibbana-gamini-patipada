### ဩဓိသောဖရဏာ မေတ္တာ စေတောဝိမုတ္တိ (၇) မျိုး

- ၁။ သဗ္ဗာ ဣတ္ထိယော (က) အဝေရာ ဟောန္တု၊
  - ( ခ ) အဗျာပဇ္ဇာ ဟောန္တု၊
  - ( ဂ ) အနီဃာ ဟောန္တု၊
  - (ဃ) သုခီ အတ္တာနံ ပရိဟရန္တု။
- ၂။ သဗ္ဗေ ပုရိသာ ။ ပ ။
- ၃။ သဗ္ဗေ အရိယာ ။ ပ ။
- ၄။ သဗ္ဗေ အနရိယာ ။ ပ ။
- ၅။ သဗ္ဗေ ဒေဝါ ။ ပ ။
- ၆။ သဗ္ဗေ မနုဿာ ။ ပ ။
- ၇။ သဗ္ဗေ ဝိနိပါတိကာ (က) အဝေရာ ဟောန္တု၊
  - ( ခ ) အဗျာပဇ္ဇာ ဟောန္တု၊
  - ( ဂ ) အနီဃာ ဟောန္တု၊
  - (ဃ) သုခီ အတ္တာနံ ပရိဟရန္တု။

- ၁။ ခပ်သိမ်းကုန်သော အမျိုးသမီးတို့သည် -----
  - (က) ဘေးရန်ကင်းကြပါစေ၊
  - ( ခ ) စိတ်ဆင်းရဲကင်းကြပါစေ၊
  - ( ဂ ) ကိုယ်ဆင်းရဲကင်းကြပါစေ၊
  - (ဃ) ကျန်းကျန်းမာမာ ချမ်းချမ်းသာသာဖြင့် မိမိတို့၏ ခန္ဓာဝန်ကို ရွက်ဆောင်နိုင်ကြပါစေ။
- ၂။ ခပ်သိမ်းကုန်သော အမျိုးသားတို့သည်။ ပ ။
- ၃။ ခပ်သိမ်းကုန်သော အရိယာသူတော်ကောင်းတို့သည်။ ပ ။
- ၄။ ခပ်သိမ်းကုန်သော အရိယာမဟုတ်သော ပုထုဇန်တို့သည်။ ပ ။
- ၅။ ခပ်သိမ်းကုန်သော နတ် (ဗြဟ္မာ) တို့သည်။ ပ ။
- ၆။ ခပ်သိမ်းကုန်သော လူသားတို့သည်။ ပ ။
- ၇။ ခပ်သိမ်းကုန်သော အပါယ်သား သတ္တဝါတို့သည်။ ပ ။

အနောဓိသောဖရဏာ မေတ္တာ စေတောဝိမုတ္တိ (၅)မျိုး， ဩဓိသောဖရဏာ မေတ္တာ စေတောဝိမုတ္တိ (၇)မျိုးနှစ်ရပ်ပေါင်းသော် (၁၂)မျိုးတည်း။ 
တစ်မျိုးတစ်မျိုး၌ ဘေးရန်ကင်းကြပါစေ - စသည်ဖြင့် (၄)ချက် (၄)ချက်စီမေတ္တာပွားရသဖြင့် (၄ _ ၁၂ = ၄၈) မျိုး ဖြစ်သည်။
