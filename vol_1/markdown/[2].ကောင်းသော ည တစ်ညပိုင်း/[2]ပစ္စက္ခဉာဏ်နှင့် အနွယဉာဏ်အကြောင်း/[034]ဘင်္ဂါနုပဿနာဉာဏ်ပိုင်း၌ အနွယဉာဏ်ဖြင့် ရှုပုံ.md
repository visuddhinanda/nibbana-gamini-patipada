### ဘင်္ဂါနုပဿနာဉာဏ်ပိုင်း၌ အနွယဉာဏ်ဖြင့် ရှုပုံ

အထ ဝါ သော ဧဝံ ဝိရတ္တော ယထာဒိဋ္ဌံ သင်္ခါရဂတံ၊ တထာ အဒိဋ္ဌမ္ပိ အနွယဉာဏ၀သေန နိရောဓေတိ၊ နော သမုဒေတိ။ 
နိရောဓတော၀ မနသိကရောတိ။ 
နိရောဓမေဝဿ ပဿတိ၊ နော သမုဒယန္တိ အတ္ထော။ (ဝိသုဒ္ဓိ၊၂၊၂၇၉။)

**ဧဝံ ဝိရတ္တော**တိ ဧဝံ ဘင်္ဂါနုပဿနာနုသာရေန ဝိရတ္တော။ 
ယထာ ဒိဋ္ဌံ သမ္ပတိ ဥပဋ္ဌိတံ သင်္ခါရဂတံ နိရောဓေတိ နိရောဓံ မနသိကရောတိ။ 
အဒိဋ္ဌမ္ပိ အတီတာနာဂတံ အနွယဉာဏ၀သေန ယထာ ဣဒံ ဧတရဟိ၊ ဧဝံ ဣတရေပီတိ အနုမိနန္တော နိရောဓေတိ မနသိကတဿာပိ နိရောဓံ ကရောတိ၊ **နော သမုဒေတီ**တိ ဧတ္ထာပိ ဧသေ၀ နယော၊ နော သမုဒယံ မနသိကရောတီတိ အတ္ထော။ (မဟာဋီ၊၂၊၄၄၂။)

အထ ဝါ = တစ်နည်းသော်ကား။ 
ဧဝံ = ဤဘင်္ဂါနုပဿနာကို လျှောက်သဖြင့် (= ဤဘင်္ဂါနုပဿနာ- ဉာဏ်သို့ အစဉ်လျှောက်သဖြင့် )။ 
ဝိရတ္တော = တပ်ခြင်းကင်းသော။ 
သော ယောဂါ၀စရော = ထိုယောဂါ၀စရသည်။ 
ဒိဋ္ဌံ = မျက်မှောက်မြင်အပ်သော၊ဝါ၊ ရှေးရှူထင်လာသော။ 
သင်္ခါရဂတံ = သင်္ခါရတရားကို။ 
နိရောဓေတိ နိရောဓံ မနသိကရောတိ ယထာ = ချုပ်ခြင်းကို နှလုံးသွင်းသကဲ့သို့။ 
တထာ = ထို့အတူ။ 
အဒိဋ္ဌမ္ပိ = မမြင်အပ်သော အတိတ် အနာဂတ် သင်္ခါရတရားကိုလည်း။ 
အနွယဉာဏ၀သေန = အလျော်ဆသော ဉာဏ်၏ အစွမ်းဖြင့်။ 
(ဧတရဟိ = ယခုမျက်မှောက်၌။ ဣဒံ ယထာ = ဤပစ္စုပ္ပန် ဒိဋ္ဌသင်္ခါရကဲ့သို့။ ဧဝံ တထာ = ထို့အတူ။ တမ္ပိ = ထိုအဒိဋ္ဌဖြစ်သော အတိတ် အနာဂတ် သင်္ခါရတရားသည်လည်း။ ဧဝံ သဘာ၀မေ၀ = ဤသို့ ချုပ်ပျက်ခြင်း သဘော ရှိသည်သာတည်း။ ဣတိ = ဤသို့။ အနုမိနန္တော = နှိုင်းရှည့်လျက်။) 
နိရောဓေတိ = နိရောဓံ မနသိကရောတိ = နိရောဓကို နှလုံးသွင်း၏။ 
နော သမုဒေတိ = နော သမုဒယံ မနသိကရောတိ = သမုဒယကို နှလုံးမသွင်း။ 
နိရောဓတော၀ = ချုပ်သောအားဖြင့် သာလျှင်။ 
မနသိကရောတိ = နှလုံးသွင်း၏။ 
အဿ သင်္ခါရဂတဿ = ထို သင်္ခါရတရား၏။ 
နိရောဓမေ၀ = နိရောဓကို သာလျှင်။ 
ပဿတိ = ရှု၏။ 
သမုဒယံ = သမုဒယကို။ 
နော ပဿတိ = မရှု။ (ပြည်-ဝိသုဒ္ဓိမဂ်နိဿယ၊၅၊၂၀၇-၂၀၈။)

= တစ်နည်း --- ဤ ဘင်္ဂါနုပဿနာဉာဏ်သို့ အစဉ်လျှောက်သဖြင့် သင်္ခါရတရားတို့ အပေါ်၌ (ဝိက္ခမ္ဘန- ပဟာန်ဖြင့် ရာဂကို ခပ်ကြာကြာ ခွာထားနိုင်သောအားဖြင့်) တပ်မက်ခြင်း ရာဂကင်းသော ထိုယောဂါ၀စရ- ပုဂ္ဂိုလ်သည် ဘင်္ဂါနုပဿနာဉာဏ်ဖြင့် မျက်မှောက်သိမြင်အပ်သော， ဘင်္ဂါနုပဿနာဉာဏ်၀၌ ရှေးရှူ ထင်လာ- သော သင်္ခါရတရား အပေါင်းကို ယင်း သင်္ခါရတရား အပေါင်း၏ ချုပ်ခြင်းကို နှလုံးသွင်းသကဲ့သို့ ထို့အတူ ဘင်္ဂါနုပဿနာဉာဏ်ဖြင့် မတွေ့ မမြင် အပ်သေးသော အတိတ် အနာဂတ် သင်္ခါရတရား အပေါင်းကိုလည်း အလျော်ဆ၍ ယူတတ်သော **အနွယဉာဏ်**၏ အစွမ်းဖြင့် ---- “ယခုမျက်မှောက် ဘင်္ဂါနုပဿနာဉာဏ်တည်း ဟူသော လက်ငင်း **ပစ္စက္ခဉာဏ်**ဖြင့် မြင်အပ်သော ဤသင်္ခါရတရားကဲ့သို့ ထို့အတူ ဘင်္ဂါနုပဿနာဉာဏ်တည်း ဟူသော ဝိပဿနာဉာဏ်ဖြင့် မမြင်အပ်သေးသော အတိတ် အနာဂတ် သင်္ခါရတရားသည်လည်း ဤသို့ချုပ်ခြင်း သဘောရှိသည်သာတည်း ” ---- ဟု ဤသို့ နှိုင်းရှည့်လျက် ချုပ်ခြင်း နိရောဓသဘောကို နှလုံးသွင်း၏၊ ဖြစ်ခြင်း သမုဒယကို နှလုံးမသွင်း။ 
ချုပ်သောအားဖြင့်သာလျှင် = အချုပ်ကိုသာလျှင် နှလုံးသွင်း၏၊ ထို သင်္ခါရတရား- အပေါင်း၏ ချုပ်ခြင်း နိရောဓကိုသာလျှင် ရှု၏၊ ဖြစ်ခြင်း သမုဒယကို မရှု။ 
ဤကား အထက်ပါ ပဋိသမ္ဘိဒါမဂ္ဂ ပါဠိတော်၏ ဆိုလိုရင်းတည်း။ (ဝိသုဒ္ဓိ၊၂၊၂၇၉။ မဟာဋီ၊၂၊၄၄၂။)

အာရမ္မဏအနွယေန၊ ဥဘော ဧက၀ဝတ္ထနာ။

နိရောဓေ အဓိမုတ္တတာ၊ ၀ယလက္ခဏဝိပဿနာ။ (ပဋိသံ၊၅၆။)

အာရမ္မဏအနွယေန = လက်ငင်းမျက်မှောက် သိမြင်အပ်သော ဒိဋ္ဌဖြစ်သော အာရုံကို အစဉ်လျှောက်- သဖြင့်။ 
ဥဘော = ဒိဋ္ဌ-အဒိဋ္ဌ အာရုံ နှစ်မျိုးတို့ကို။ 
ဧက၀ဝတ္ထနာ = ခဏဘင်ဟူသော တစ်ခုတည်းသော သဘော အားဖြင့် ပိုင်းခြားမှတ်သားခြင်းလည်းကောင်း။ 
နိရောဓေ = သင်္ခါရတရားတို့၏ ခဏမစဲ တသဲသဲ ချုပ်ပျက်ခြင်း ခဏိကနိရောဓ၌။ 
အဓိမုတ္တတာ = ညွတ်ကိုင်းရှိုင်းသော အဖြစ်လည်းကောင်း။ 
ဧသာ = ဤသည်ကား။ 
၀ယ- လက္ခဏဝိပဿနာ = သင်္ခါရတရားတို့၏ ပျက်ခြင်း ၀ယလက္ခဏာကို ရှုသော ဝိပဿနာ မည်၏။  

**အာရမ္မဏအနွယေန ဥဘော ဧက၀ဝတ္ထနာ**တိ ပစ္စက္ခတော ဒိဋ္ဌဿ အာရမ္မဏဿ အနွယေန အနုဂမနေန ယထာ ဣဒံ၊ တထာ အတီတေပိ သင်္ခါရဂတံ ဘိဇ္ဇိတ္ထ၊ အနာဂတေပိ ဘိဇ္ဇိဿတီတိ ဧဝံ ဥဘိန္နံ ဧကသဘာဝေ- နေ၀ ၀ဝတ္ထာပနန္တိ အတ္ထော။

ဝုတ္တမ္ပိ စေတံ ပေါရာဏေဟိ ---

“သံဝိဇ္ဇမာနမှိ ဝိသုဒ္ဓဒဿနော၊

တဒနွယံ နေတိ အတီတနာဂတေ။

သဗ္ဗေပိ သင်္ခါရဂတာ ပလောကိနော၊

ဥဿာ၀ဗိန္ဒူ သူရိယေ၀ ဥဂ္ဂတေ”တိ။ (ဝိသုဒ္ဓိ၊၂၊၂၈၀။)
 
**ဥဘော**တိ ဒိဋ္ဌာဒိဋ္ဌဘာဝေန ဒွေပိ အာရမ္မဏာနိ။ 
**ဧက၀ဝတ္ထနာ**တိ ခဏဘင်္ဂုရတာယ သမာနာတိ ၀ဝတ္ထာပနာ။ ။ 
**သံဝိဇ္ဇမာနမှီ**တိ ပစ္စက္ခတော ဥပလဗ္ဘမာနေ ပစ္စုပ္ပန္နေ။ 
**ဝိသုဒ္ဓဒဿနော**တိ ဘင်္ဂဒဿနဿ ပရိဗန္ဓဝိဓမနေန ဝိသုဒ္ဓဉာဏော၊ သုဝိသုဒ္ဓဘင်္ဂဒဿနောတိ အတ္ထော။ 
**တဒနွယံ နေတီ**တိ တဿ ပစ္စုပ္ပန္နသင်္ခါရ- ဘင်္ဂဒဿိနော ဉာဏဿ အနွယံ အနုဂတဘူတံ ဉာဏံ ပ၀တ္တေတိ။ 
ကတ္ထ ပန ပ၀တ္တေတီတိ အာဟ “**အတီတ-** **နာဂတေ”**တိ။ 
အတီတေ， အနာဂတေ စ သင်္ခါရဂတေ။ 
**သဗ္ဗေပီ**တိအာဒိ တဿ ပဝတ္တနာကာရဒဿနံ။ (မဟာဋီ၊၂၊၄၄၃။)

ဉာတဉ္စ ဉာဏဉ္စ ဥဘောပိ ဝိပဿတိ။ (ဝိသုဒ္ဓိ၊၂၊၂၇၈။)

**အာရမ္မဏအနွယေန ဥဘော ဧက၀ဝတ္ထနာ** --- ပစ္စက္ခအားဖြင့် (= ပစ္စက္ခဉာဏ်ဖြင့်) ကိုယ်တိုင်မျက်- မှောက် ထွင်းဖောက် သိမြင်အပ်ပြီးသော ရှုမြင်အပ်ပြီးသော ဉာတအမည်ရသော ဒုက္ခသစ္စာတရား သမုဒယသစ္စာ တရား， ဉာဏအမည်ရသော (လောကီမဂ္ဂသစ္စာတရားတည်းဟူသော) ဝိပဿနာဉာဏ် --- ဤ ဉာတဉာဏ- အမည်ရသော သင်္ခါရအာရုံကို အစဉ်လျှောက်သဖြင့်， ဘင်္ဂါနုပဿနာဉာဏ်ဖြင့် လက်ငင်းဒိဋ္ဌ သိမြင်အပ်သော ဤပစ္စုပ္ပန္နဓမ္မ သင်္ခါရတရားသည် ပျက်သကဲ့သို့ ထို့အတူ အတိတ်၌လည်း သင်္ခါရတရားအပေါင်းသည် ပျက်ခဲ့ပြီ၊ နောင်အနာဂတ်၌လည်း ပျက်လတ္တံ့ဟု --- ဝိပဿနာဉာဏ်ဖြင့် ရှုမြင်အပ်ပြီးသော ဒိဋ္ဌအာရုံ， ဝိပဿနာဉာဏ်ဖြင့် မရှုမြင်အပ်သေးသော အဒိဋ္ဌအာရုံဟူသော နှစ်ပါးကုန်သော အာရုံတို့ကို ဤခဏဘင်အားဖြင့် ခဏမစဲ တသဲသဲ ပျက်ခြင်းဟူသော တူသောသဘောအားဖြင့်သာလျှင် ကောင်းစွာ မှတ်သားခြင်းကို ပြုလုပ်အပ်ပေသည်။ 
ဤကား ဆိုလိုရင်း အနက်သဘောတည်း။ 
မှန်ပေသည် --- ရှေးရှေးအဋ္ဌကထာဆရာတော်တို့သည် ဤစကားကို မိန့်ဆို- တော်မူကြကုန်၏။ ---

သံဝိဇ္ဇမာနမှိ ။ ပ ။ ဥဂ္ဂတေ ။ (အထက်တွင် ကြည့်ပါ။)

သံဝိဇ္ဇမာနမှိ = ပစ္စက္ခအားဖြင့်， ရအပ်သော ပစ္စက္ခဉာဏ်ဖြင့် သိမြင်အပ်သော ပစ္စုပ္ပန္နဓမ္မ သင်္ခါရ- တရားအပေါင်း၌ (= ဒိဋ္ဌတရားဟူလိုသည်)။ 
ဝိသုဒ္ဓဒဿနော = အထူးသဖြင့် စင်ကြယ်သော， သင်္ခါရတရားတို့၏ ဘင်ကို ရှုသော ဘင်္ဂါနုပဿနာဉာဏ်အမြင်ရှိသော ယောဂါ၀စရပုဂ္ဂိုလ်သည်။ 
တဒနွယံ = ထိုပစ္စုပ္ပန္နဓမ္မ သင်္ခါရတရား၏ ပျက်ခြင်းဘင်ကို ရှုသော ဘင်္ဂါနုပဿနာဉာဏ်သို့ အစဉ်လိုက်၍ ဖြစ်သော ဝိပဿနာဉာဏ်ကို။ 
အတီတေ = အတိတ် ဖြစ်သော သင်္ခါရတရား၌ လည်းကောင်း။ 
အနာဂတေ = အနာဂတ် ဖြစ်သော သင်္ခါရတရား၌လည်းကောင်း။ 
နေတိ = ဆောင်၏ = ဖြစ်စေ၏။ 
သဗ္ဗေပိ = ကြွင်းမဲ့ဥဿုံ အလုံးစုံလည်း- ဖြစ်ကုန်သော။ 
သင်္ခါရဂတာ = အတိတ်သင်္ခါရ အနာဂတ်သင်္ခါရ ပစ္စုပ္ပန်သင်္ခါရ တရားအပေါင်းတို့သည်။ 
ပလောကိနော = ပျက်ခြင်းသဘော ရှိကုန်၏။ 
ကိမိ၀ = အဘယ်ကဲ့သို့နည်းဟူမူကား --- သူရိယေ = နေသည်။ 
ဥဂ္ဂတေ = တက်လတ်သည် ရှိသော်။ 
ဥဿာ၀ဗိန္ဒု = ဆီးနှင်းပေါက်သည်။ 
ပလောကတိဣ၀ = ပျောက်ပျက်- သကဲ့သို့တည်း။ (ဝိသုဒ္ဓိ၊၂၊၂၈၀။)

= ပစ္စက္ခအားဖြင့် ရအပ်သော ပစ္စက္ခဉာဏ်ဖြင့် သိမြင်အပ်သော ဒိဋ္ဌအမည်ရသော ပစ္စုပ္ပန္နဓမ္မ သင်္ခါရ- တရားအပေါင်း၌ ပျက်ခြင်းဘင်ကို ရှုသော အထူးသဖြင့် စင်ကြယ်သော ဘင်္ဂါနုပဿနာဉာဏ်အမြင်ရှိသော ယောဂါ၀စရပုဂ္ဂိုလ်သည် ထိုပစ္စုပ္ပန္နဓမ္မ သင်္ခါရတရား၏ ဘင်ကို ရှုသော ဘင်္ဂါနုပဿနာဉာဏ်သို့ အစဉ်လိုက်၍ ဖြစ်သော ဝိပဿနာဉာဏ်ကို အတိတ်သင်္ခါရတရား အနာဂတ်သင်္ခါရတရား၌ နည်းကို ဆောင်၏၊ ဝိပဿနာ- ဉာဏ်ကို ဖြစ်စေ၏။ 
ကြွင်းမဲ့ဥဿုံ အလုံးစုံလည်း ဖြစ်ကုန်သော အတိတ် အနာဂတ် ပစ္စုပ္ပန် သင်္ခါရတရား မှန်သမျှတို့သည် ပျက်ခြင်းသဘော ရှိကုန်၏။ 
အဘယ်ကဲ့သို့နည်းဟူမူကား -- နေခြည်ထိပျောက် ဆီးနှင်းပေါက် ကဲ့သို့တည်း။ (ဝိသုဒ္ဓိ၊၂၊၂၈၀။ မဟာဋီ၊၂၊၄၄၃။)

**နိရောဓေ အဓိမုတ္တတာ = ခဏနိရောဓ၌ ညွတ်ကိုင်းရှိုင်းမှု**

**နိရောဓေ အဓိမုတ္တတာ**တိ ဧဝံ ဥဘိန္နံ ဘင်္ဂ၀သေန ဧက၀ဝတ္ထာနံ ကတွာ တသ္မိညေ၀ ဘင်္ဂသင်္ခါတေ နိရောဓေ အဓိမုတ္တတာ တဂ္ဂရုတာ တန္နိန္နတာ တပ္ပေါဏတာ တပ္ပဗ္ဘာရတာတိ အတ္ထော။ 
**၀ယလက္ခဏဝိပ-** **ဿနာ**တိ ဧသာ ၀ယလက္ခဏဝိပဿနာ နာမာတိ ဝုတ္တံ ဟောတိ။ (ဝိသုဒ္ဓိ၊၂၊၂၈၀။)

**ဘင်္ဂသင်္ခါတေ နိရောဓေ**၊ န နိဗ္ဗာနသင်္ခါတေတိ အဓိပ္ပါယော။ 
**ဧသာ**တိ ယာ အာရမ္မဏနွယေန ဥဘော ဒိဋ္ဌာဒိဋ္ဌေ ခဏဘင်္ဂုရတာယ ဧကဘာဝေန ၀ဝတ္ထာပနာ၊ ဧသာ ၀ယလက္ခဏေ ဝိပရိဏာမလက္ခဏေ **ဝိပဿနာ** ဘင်္ဂါနုပဿနာတိ အတ္ထော။ (မဟာဋီ၊၂၊၄၄၃။)
